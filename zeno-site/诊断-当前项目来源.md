## 六、为什么浏览器看不到变化

**结论：代码没有问题，问题在部署/访问方式。**

可能原因：

### 原因 A（最可能）：你在 zenoaihome.com 看，但域名还指向旧 WordPress
- DNS 尚未切换到 Vercel
- 改代码对 zenoaihome.com 没有任何效果，因为那个网站根本不是这个项目在跑

### 原因 B：代码没有推送到 Vercel
- 本地改了代码，但没有 git commit + git push
- Vercel 没有触发重新部署

### 原因 C：本地 dev server 未启动
- 需要手动运行 `npm run dev`，然后打开 `http://localhost:3000`
