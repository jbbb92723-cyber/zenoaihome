# 管理员识别配置说明

本文档说明如何在 Vercel 上配置管理员账号，让你在登录后能够使用草稿推送、图片生成等管理员专属功能。

---

## 背景

网站使用 `ADMIN_EMAILS` 和 `ADMIN_USER_IDS` 两个环境变量来识别管理员身份。

- **`ADMIN_EMAILS`**：按登录邮箱识别。适用于 OAuth 提供商返回真实邮箱的情况。
- **`ADMIN_USER_IDS`**：按用户 ID 识别。适用于 OAuth 提供商返回隐私邮箱（如 IDC Flare）的情况。**更稳定，推荐优先使用。**

两个变量同时支持，任意一个匹配即视为管理员。

---

## 获取你的邮箱或用户 ID

1. 打开网站，点击右上角头像或导航中的「登录」按钮
2. 完成登录（IDC Flare 账号）
3. 登录后，访问 `/account`（个人中心）
4. 在「身份信息」表格中，你会看到：
   - **登录邮箱**：OAuth 实际返回的邮箱（可能是隐私邮箱）
   - **用户 ID**：OAuth 返回的唯一 ID，点击「复制」按钮即可复制

---

## 配置步骤

### 方法一：按用户 ID 配置（推荐）

1. 按上方说明获取你的**用户 ID**，复制
2. 登录 [Vercel 控制台](https://vercel.com)
3. 进入 `zenoaihome` 项目 → **Settings** → **Environment Variables**
4. 找到 `ADMIN_USER_IDS`（如没有，点击「Add New」新增）
5. 将用户 ID 粘贴进去，点击 Save
6. 返回项目首页，点击 **Deployments** → **Redeploy**（选最新一次部署）

### 方法二：按邮箱配置

1. 按上方说明获取你的**登录邮箱**，复制
2. 同上，找到 `ADMIN_EMAILS` 并填入邮箱，Save
3. Redeploy

> **注意**：IDC Flare 通常返回形如 `xxxxxxxx@idcflare.com` 的隐私邮箱，而不是你的真实 QQ 邮箱。如果你在 ADMIN_EMAILS 中填的是 `zenoaihome@qq.com` 但无法识别为管理员，请改用用户 ID 方法。

---

## 验证是否成功

Redeploy 完成后：
1. 重新登录网站
2. 访问 `/tools/md2wechat/status`
3. 在「管理员权限」区域查看：
   - **当前用户是否管理员**：是 ✓

如果仍显示「否」，检查：
- 用户 ID 或邮箱是否复制完整，有无多余空格
- Vercel 环境变量是否保存成功（去 Environment Variables 页面确认）
- 是否已 Redeploy（修改环境变量不会自动生效）

---

## 多管理员

`ADMIN_EMAILS` 和 `ADMIN_USER_IDS` 均支持逗号分隔的多个值：

```
ADMIN_EMAILS=admin@example.com,another@example.com
ADMIN_USER_IDS=abc123,def456
```
