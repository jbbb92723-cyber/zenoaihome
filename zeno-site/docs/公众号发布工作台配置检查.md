> 写给非程序员看的配置指南。按顺序读完，你就能自己排查所有问题。

## 2. 如果 /tools/md2wechat 打不开怎么办

**步骤一：** 检查 Vercel 是否部署成功。
打开 [https://vercel.com/dashboard](https://vercel.com/dashboard)，查看最新一次 Deployment 是否为绿色 ✅。

**步骤二：** 如果 Vercel 显示红色报错，点击报错查看 Build Log，把错误信息截图告诉开发者。

**步骤三：** 如果部署成功但页面打开空白/500错误，前往：
```
https://zenoaihome.com/tools/md2wechat/status
```
查看配置状态页面，确认哪项 API 未配置。

---

## 3. Vercel 环境变量怎么填

1. 打开 [https://vercel.com/dashboard](https://vercel.com/dashboard)
2. 点击你的项目 → **Settings** → **Environment Variables**
3. 每个变量填写：名称（Name）、值（Value），选 **Production**（或 All Environments）
4. 填完后点击 **Save**

⚠️ **重要：填完环境变量后必须 Redeploy 才生效。**

---

## 4. 填完环境变量为什么要 Redeploy

Vercel 的环境变量在 **构建时** 注入到服务器代码中。填写后旧的部署不会自动更新，必须重新部署才能读取新值。

**操作方法：**
- Vercel Dashboard → Deployments → 最新一条 → 右上角 **Redeploy**

## 5. md2wechat 需要哪些参数

| 变量名 | 说明 | 哪里获取 |
|--------|------|----------|
| `MD2WECHAT_BASE_URL` | API 服务地址，例如 `https://md2wechat.cn` | md2wechat 控制台 |
| `MD2WECHAT_API_KEY` | API 密钥 | md2wechat 控制台 → API Key |
| `MD2WECHAT_CONVERT_ENDPOINT` | 可选，转换接口路径，默认 `/api/v1/convert` | 留空即可 |

**注意：** `MD2WECHAT_BASE_URL` 填写域名，不要加末尾斜杠，不要加 `/api/v1/...`。

---

## 6. 火山引擎 Doubao-Seedream-5.0-lite 需要哪些参数

| 变量名 | 说明 | 哪里获取 |
|--------|------|----------|
| `VOLCENGINE_ARK_API_KEY` | 火山方舟 API Key | [控制台](https://console.volcengine.com/ark) → API Key |
| `VOLCENGINE_IMAGE_MODEL` | 模型 ID，默认填 `Doubao-Seedream-5.0-lite` | 控制台确认后填入 |
| `VOLCENGINE_IMAGE_BASE_URL` | API 基础地址，默认 `https://ark.cn-beijing.volces.com/api/v3` | 一般不用改 |
| `VOLCENGINE_IMAGE_PRICE_PER_IMAGE` | 单张预估价格（元），仅用于前端显示，默认 `0.22` | 根据控制台计费页调整 |

**注意：** 图片生成功能**只有管理员**可以使用，普通用户调用会返回 403 错误，不会产生费用。

## 7. 微信公众号草稿箱需要哪些参数

| 变量名 | 说明 | 哪里获取 |
|--------|------|----------|
| `WECHAT_APPID` | 公众号 AppID | 微信公众平台 → 设置 → 基本配置 |
| `WECHAT_APP_SECRET` | 公众号 AppSecret | 同上，需要管理员扫码确认 |
| `WECHAT_DEFAULT_AUTHOR` | 默认作者名，可填 `Zeno` | 自己填写 |
| `WECHAT_DEFAULT_COVER_MEDIA_ID` | 默认封面图 media_id（可选） | 见下方说明 |

## 8. 默认封面 media_id 怎么理解

`media_id` 是微信内部的图片素材 ID。你上传一张图片到微信公众号素材库后，微信会给这张图片分配一个 `media_id`。

**如何获取：**
1. 登录微信公众平台 → **素材管理** → **图片与文字素材** → 上传一张封面图
2. 通过微信 API 接口查询 media_id（需要开发者配合）
3. 或者推送草稿时不填封面，在公众号后台手动添加

目前版本可以**不填**此字段，草稿仍然可以创建，只是没有默认封面。

## 9. IP 白名单为什么重要

微信公众号 API 要求调用方的服务器 IP 在白名单里，否则会返回 40164 错误。

**Vercel 的 IP 是动态的，不固定。**

**解决方案（二选一）：**

1. **通过代理转发：** 使用 md2wechat 的草稿接口，让 md2wechat 服务器调用微信 API（md2wechat 的 IP 已加入白名单）。
2. **固定 IP 方案：** 使用 Cloudflare Workers 或其他有固定 IP 的服务代理请求，然后把该 IP 加入微信白名单。

当前工作台走的是 md2wechat 草稿接口（方案 1），推荐使用此方式。

## 10. 管理员邮箱 ADMIN_EMAILS 怎么填

`ADMIN_EMAILS` 填写有管理员权限的账号邮箱，多个邮箱用英文逗号分隔：

```
ADMIN_EMAILS=your@email.com,other@email.com
```

- 登录该邮箱账号后，页面会显示管理员区域（AI 配图、推送草稿）
- 未登录或邮箱不在列表中，管理员区域不可见
- 如果 `ADMIN_EMAILS` 为空，**任何人都没有管理员权限**

## 11. 如何判断 API 是否成功

访问配置状态页面：
```
https://zenoaihome.com/tools/md2wechat/status
```

页面会显示每项配置的状态（已配置 / 未配置），以及测试按钮。

- **md2wechat：** 点击「测试 Markdown 转换」，成功则显示字数和 HTML 长度
- **火山引擎：** 管理员登录后点击「测试图片生成」（有费用，需二次确认）
- **微信草稿箱：** 点击「检查微信配置完整性」，只做本地参数检查，不产生实际请求

## 12. 一步步测试流程

### 第一步：确认页面可以打开

访问：`https://zenoaihome.com/tools/md2wechat`

看到 Markdown 编辑框就代表页面正常。

### 第二步：测试 Markdown 转换

1. 在编辑框输入一些 Markdown 文字
2. 点击「转换」按钮
3. 右侧出现预览说明转换成功
4. 如果报错「md2wechat API 尚未配置」，需要去 Vercel 填写 `MD2WECHAT_BASE_URL` 和 `MD2WECHAT_API_KEY`

### 第三步：测试图片生成（管理员）

1. 用管理员邮箱账号登录网站
2. 打开排版工具页面，看是否出现「管理员功能」区块
3. 输入文章内容后点击「生成封面图」
4. 如果报错「火山引擎图片生成 API 尚未配置」，需要填写 `VOLCENGINE_ARK_API_KEY` 和 `VOLCENGINE_IMAGE_MODEL`

### 第四步：测试草稿箱配置

1. 访问：`https://zenoaihome.com/tools/md2wechat/status`
2. 查看「微信公众号草稿箱」部分是否显示「已配置」
3. 点击「检查微信配置完整性」

## 常见问题

**Q：转换后复制 HTML，粘贴到公众号为什么样式不对？**
建议使用「默认」或「苹果范」主题，部分主题依赖外部字体在公众号内可能不完全生效。粘贴后在公众号编辑器里预览，确认无误后再发布。

**Q：推送草稿后在哪里能看到？**
登录微信公众平台 → 内容 → 草稿箱。找到对应标题的草稿，审核后手动点击发布。

**Q：推送草稿会自动发布吗？**
不会。系统只创建草稿，不会自动群发。所有发布操作必须由你在公众号后台手动确认。

**Q：管理员区域不显示怎么办？**
1. 确认已登录管理员邮箱账号
2. 确认 Vercel 中 `ADMIN_EMAILS` 已填写且与登录邮箱一致
3. 填写后 Redeploy 一次
4. 访问配置状态页面检查「当前用户识别为管理员」是否显示「已配置」
