## 3. 核心表格设计（六张）

### 表 1：报价单完整度检查表

- **用途**：判断这份报价单"该有的项目"是否都列了。
- **使用方法**：按空间（客厅/卧室/厨房/卫生间/阳台/全屋）逐区核对，每个区域勾选实际出现的施工项。
- **字段**：空间分区 / 施工项目 / 是否出现 / 单位 / 单价 / 备注 / 风险等级
- **风险判断**：单个空间施工项 < 3 项 = 高风险；缺关键工序（如卫生间无防水、厨房无墙地砖）= 直接退回重报

### 表 2：漏项风险检查表

- **用途**：识别报价单"应该有但没写"的项目。
- **使用方法**：对照"标准漏项清单"逐项确认是否在报价范围内。
- **字段**：常见漏项 / 报价是否包含 / 不含的话怎么算 / 预计追加金额 / 是否能书面确认
- **风险判断**：3 项以上"未确认"= 高风险；任意一项写"另计/暂估/项"= 黄色预警

### 表 3：模糊描述识别表

- **用途**：标记报价单里"看似详细、实际模糊"的描述。
- **使用方法**：每行描述按"工艺标准是否清晰、材料品牌是否明确、计量单位是否合理"三个维度打分。
- **字段**：原文描述 / 工艺标准 / 材料品牌 / 计量单位 / 风险等级 / 应该追问什么
- **风险判断**：三项中任意两项"模糊"= 高风险

### 表 4：增项风险记录表

- **用途**：在多家报价比较时记录"哪些项被拆出来另算"。
- **使用方法**：把所有"另计/甲供/暂估"项集中列出，估算最坏情况下追加金额。
- **字段**：项目 / 报价方处理方式 / 另算金额估算 / 是否可在合同前固定 / 备注
- **风险判断**：另算金额 > 总价 15% = 高风险；多家报价同一项处理方式不一致 = 必问

### 表 5：水电报价检查表

- **用途**：水电是装修中最容易超支的隐蔽工程，单独检查。
- **使用方法**：按"强电/弱电/给水/排水/防水"五个维度核对。
- **字段**：项目 / 计量单位 / 单价区间参考 / 报价是否在区间内 / 是否含开槽/封槽 / 风险等级
- **风险判断**：单价偏离市场区间 30% 以上 = 警惕；按"项"或"户"计价而非"米/平米" = 高风险

### 表 6：合同条款检查表

- **用途**：签合同前最后一次自查。
- **使用方法**：按"付款节点 / 工期约定 / 增项规则 / 验收标准 / 违约责任 / 售后期限"六类逐项核对。
- **字段**：条款类别 / 合同原文 / 是否合理 / 风险点 / 修改建议
- **风险判断**：首付比例 > 40%、增项无上限、验收标准模糊任一出现 = 不建议签
