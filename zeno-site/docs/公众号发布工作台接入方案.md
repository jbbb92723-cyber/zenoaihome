# 公众号发布工作台接入方案

## 1. 工具定位

这是一个内嵌在 Zeno AI Home 网站里的轻量级发布工具，面向个人写作工作流。核心逻辑：写好文章 → 在网站内转换排版 → 一键推送草稿到微信公众号后台 → 人工审核发布。

不是面向所有用户的公开服务，也不做自动群发。

---

## 2. 公开用户能做什么

访问 `/tools/md2wechat`：

- 输入 Markdown 内容
- 选择主题（48 种）和字体大小
- 点击「转换」获得微信排版 HTML
- 预览排版效果
- 复制 HTML 粘贴到微信公众号后台
- 复制原始 Markdown 文本

**以上功能完全免费，不需要登录。**

---

## 3. 管理员能做什么

在公开功能基础上，额外开放：

- 生成封面图（豆包图片生成）
- 生成文中配图（豆包图片生成）
- 将文章直接推送到微信公众号草稿箱（通过 md2wechat article-draft 接口）

---

## 4. md2wechat API 的作用

md2wechat（`md2wechat.cn`）是一个专门做 Markdown → 微信公众号 HTML 转换的第三方服务。

主要用途：
- `POST /api/v1/convert`：将 Markdown 转成带微信样式的 HTML，支持 48 种主题
- `POST /article-draft`：同时转换并在微信公众号后台创建图文草稿（需要 Wechat-Appid + App-Secret）

为什么用它：微信公众号不支持原生 Markdown，直接粘贴效果很差。md2wechat 解决了排版问题，省去手动调格式的工作。

---

## 5. 豆包图片生成 API 的作用

用于生成文章封面图和文中配图。调用火山引擎豆包（Seedream 系列）的文生图接口。

- 封面图：推荐 `2560x1440`，适合微信公众号头图比例
- 文中配图：`1024x1024`

成本约 ¥0.22/张（以火山控制台实际价格为准）。

---

## 6. 微信草稿箱 API 的作用

通过 md2wechat 的 `article-draft` 接口，在微信公众号后台创建图文消息草稿。草稿创建后不会自动发布，需要人工到微信公众号后台审核后手动群发。

---

## 7. 为什么不把草稿箱发布开放给所有人

- 草稿创建需要 Wechat-AppSecret，这是高敏感凭证，如果开放接口容易被滥用
- 每次调用都通过管理员的公众号账号创建草稿，无法隔离用户行为
- 防止任何人都能往我的公众号后台塞内容
- v2.0 考虑支持用户绑定自己的公众号后再开放

---

## 8. 为什么图片生成要限管理员

- 豆包图片生成有实际费用（约 ¥0.22/张）
- 开放给所有人可能产生不可控的计费
- 当前没有计费追踪系统
- 后续如果做付费会员系统，可以逐步开放额度

---

## 9. 版本路线

### v1.0（当前）

- Markdown 输入、主题选择
- 调用 md2wechat 转换 HTML
- 预览排版效果
- 复制 HTML / 复制原文

### v1.1（下一步）

- 管理员生成封面图（豆包图片生成）
- 管理员生成文中配图
- 图片 URL 可用于草稿封面

### v1.2（再下一步）

- 管理员将文章推送到微信公众号草稿箱
- 支持自定义封面图（上传或填写 URL）
- 草稿推送结果提示

### v2.0（未来）

- 用户绑定自己的公众号 AppID + AppSecret
- 用户自己的草稿箱推送，与管理员账号隔离
- 按用户计费图片生成额度
