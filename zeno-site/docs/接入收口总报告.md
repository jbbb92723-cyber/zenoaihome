## 五、微信公众号草稿箱

### 现状：✅ 已完整实现（双通道）

| 通道 | 文件 | 说明 |
|------|------|------|
| 通道 A：md2wechat API | `lib/md2wechat.ts` → `createWechatDraft()` | 经 md2wechat 中转，自动 HTML 转换 |

以下旧版路由使用 `ADMIN_TOKEN` 头部认证（非 session auth），保留兼容：

| 路由 | 认证方式 | 说明 |
|------|----------|------|
| `POST /api/image/generate` | `x-admin-token` | 旧版图片生成 |
| `POST /api/wechat/convert` | `x-admin-token` | 旧版 Markdown 转换 |

新版路由使用 `isAdminUser()` session 认证：

| 路由 | 认证方式 | 说明 |

| `POST /api/md2wechat/draft` | session auth | 新版草稿推送 |
| `POST /api/wechat/draft` | session auth | 新版草稿推送 |
