> 文档版本：v1.0 · 2026-04-20

## 一、用户类型

### 类型 1：未登录访客（visitor）

**谁：** 任何人，包括搜索引擎爬虫。  
**可以做什么：**
- 浏览首页、关于我、文章列表
- 阅读公开文章
- 看资料库列表（但无法领取）
- 浏览视频列表（公开视频）
- 看专题、服务页、联系页

**不能做什么：**
- 领取任何资料
- 发表评论
- 查看用户中心
- 访问会员专属内容

---

### 类型 2：已登录用户（user）

**谁：** 通过 OAuth 登录，但没有开通会员。  
**可以做什么：** 访客可以做的所有事，加上：
- 领取「登录可领」资料
- 发表评论（待审核）
- 查看用户中心（/account）
- 查看自己的下载历史

**不能做什么：**
- 领取会员专属资料
- 领取付费资料（除非单独购买）
- 访问会员专属文章

---

### 类型 3：免费会员（member）

**谁：** 开通免费会员（或邀请码开通）的登录用户。  
**可以做什么：** 已登录用户可以做的所有事，加上：
- 领取「会员可领」资料
- 阅读会员专属文章
- 访问会员专属视频

**不能做什么：**
- 领取付费资料（除非单独购买）

---

### 类型 4：付费会员（customer）

**谁：** 通过支付升级的用户。  
**可以做什么：** 会员可以做的所有事，加上：
- 下载已购买的付费资料
- 访问已购买的付费内容

---

### 类型 5：已购买单项资料用户

**谁：** 没有开通会员，但单独购买了某份资料的用户。  
**可以做什么：** 只能下载已购买的那份资料。  
**实现方式：** 查询 `orders` 表确认购买记录。

---

### 类型 6：管理员（admin）

**谁：** 赞诺本人或授权管理员。  
**可以做什么：** 所有权限，包括：
- 审核评论
- 查看管理后台（第二阶段）
- 修改用户角色
- 访问 admin 级别资源

---

## 二、资源权限级别

### public — 公开
- 任何人可以访问，不需要登录
- 搜索引擎可以索引

### login — 登录可访问
- 必须登录（任何角色用户都可以，visitor 除外）
- 用于「登录领取」类免费资料

### member — 会员可访问
- 必须是 member / customer / admin 角色
- 会员专属文章、会员专属资料

### paid — 付费可访问
- 必须已购买（customer / admin 角色）
- 单项付费资料

### admin — 管理员专属
- 只有 admin 角色
- 不对外显示，只在管理后台使用

## 五、权限判断代码

在前端，使用 `lib/permissions.ts` 中的工具函数：

```typescript
import { canAccessResource } from '@/lib/permissions'

// 判断当前用户是否可以访问某资源
const canAccess = canAccessResource(session?.user, { accessLevel: 'member' })

// 获取操作按钮文案
const buttonLabel = getAccessButtonLabel(session?.user, resource)
```

在服务端（Server Components / API Routes），通过 `auth()` 获取 session：

```typescript
import { auth } from '@/auth'

export default async function ProtectedPage() {
  const session = await auth()
  if (!session?.user) redirect('/login')
  // ...
}
```
