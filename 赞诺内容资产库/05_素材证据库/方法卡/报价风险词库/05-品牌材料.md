# 品牌材料

## 基本信息

- 风险词：品牌材料
- 常见变体：一线品牌、知名品牌、环保材料、优质材料、同档次品牌
- 常见位置：乳胶漆、水管、电线、防水、板材、瓷砖、地板、橱柜、辅材清单
- 适用场景：全包、半包、整装、主材包、辅材包
- 风险等级：高

## 一句话解释

> 报价里写“品牌材料”不等于材料已经锁定，真正要看品牌后面有没有型号、系列、规格、等级、替换规则和验收方式。

## 常见写法

```text
使用某某品牌材料
使用一线品牌辅材
使用环保材料
使用某品牌或同档次品牌
```

## 业主常见误判

- 以为写了品牌，材料就已经定好了。
- 以为大牌不同型号之间差别不大。
- 以为“同档次品牌”是客观标准。
- 材料进场时不知道怎么核对。

## 真正风险

- 同品牌低配替代。
- 只写品牌，不写系列和型号。
- 用“同档次品牌”留下替换空间。
- 环保等级、规格、厚度、功能款没有写清。
- 材料进场后缺少验收和留证方式。

## 哪些情况不是坑

“品牌材料”不是天然有问题。

合理情况包括：

- 报价中先明确品牌范围。
- 材料型号需要在深化方案后确认。
- 供应链存在缺货替换可能。

关键是：替换可以有，但替换规则和确认流程必须先写清。

## 签约前五问

1. 具体品牌、系列、型号是什么？
2. 规格、等级、环保标准写清了吗？
3. 如果缺货，能不能替换？谁确认？
4. 替换后是否不得低于原约定等级？
5. 材料进场时，我怎么验收和留证？

## 更安全的写法

### 模糊写法

```text
使用某某品牌材料
使用一线品牌辅材
```

### 更清楚的写法

```text
使用某品牌某系列某型号，规格为 XX，等级为 XX。如需替换，须经业主书面确认，且不得低于原约定等级。验收以外包装、合格证及进场确认记录为准。
```
