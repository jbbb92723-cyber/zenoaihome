# 另计 / 不含

## 基本信息

- 风险词：另计 / 不含
- 常见变体：费用另计、不包含、另行收费、按需另计、业主自理、甲方自理
- 常见位置：垃圾清运、成品保护、开孔打洞、五金配件、搬运上楼、拆后修补、远程费、管理费
- 适用场景：全包、半包、套餐装修、局改、老房翻新
- 风险等级：高

## 一句话解释

> “另计 / 不含”不是一定有问题，但如果没有写清具体不含什么、后面怎么计费、谁确认、有没有上限，就容易变成另一笔钱。

## 常见写法

```text
垃圾清运不含
五金另计
开孔费用另计
搬运上楼不含
拆后修补另计
```

## 业主常见误判

- 以为“不含”只是少一点小费用。
- 以为另计项目金额不会太大。
- 以为后期需要时再谈也来得及。
- 只看包含项，忽略不含项的累计成本。

## 真正风险

- 范围少了一截，后期必须补钱。
- 另计项目没有单价，后期议价被动。
- 没有补充清单，签约时无法计算完整预算。
- 另计前没有确认流程，施工后才告知费用。

## 哪些情况不是坑

“另计 / 不含”不是天然有问题。

合理情况包括：

- 某些个性化项目不适合放进基础报价。
- 业主可能自行采购或自行处理。
- 现场条件不同，费用确实需要另算。

关键不是不能不含，而是要先把边界和计费规则说清。

## 签约前五问

1. 具体哪些项目不含？
2. 不含项大概多少钱？
3. 是按单价还是按现场另谈？
4. 要不要先列入补充清单？
5. 超出前需不需要我确认？

## 更安全的写法

### 模糊写法

```text
垃圾清运不含
五金另计
```

### 更清楚的写法

```text
垃圾清运不含；如需另计，提前写明计费方式、单价和范围；施工前由业主确认后再执行。五金配件如不含，需在补充清单中列明。
```
