# 装修报价审核清单

> 类型：免费引流资料（内容磁铁）
> 用途：低门槛进入产品，引导进入报价单审核服务

---

## 清单定位

**用途：** 公众号关注/网站订阅时的免费资料

**问题：** 「我怎么知道这份报价单有没有坑？」

**答案：** 用这份清单，对照检查。

---

## 清单内容（草稿）

### 报价单审核清单（10条）

**报价结构类**
- [ ] 报价单是否分项（按施工阶段列清楚）？
- [ ] 是否标注了材料品牌和型号？
- [ ] 是否有工程量说明（如：「瓷砖铺贴 XX 平米」）？

**漏项类**
- [ ] 全屋灯具是否在报价单里？
- [ ] 卫生间洁具（马桶/花洒/镜柜）是否在报价单里？
- [ ] 门锁/合页/拉手等五金是否有单独说明？
- [ ] 弱电（网线/电视线/门禁）是否单独报了？

**条款类**
- [ ] 水电工程是包干还是按实结算？
- [ ] 有没有「增项需甲方书面确认」的条款？
- [ ] 有没有明确的竣工验收标准？
