# 输入收件箱

> 所有新资料先进入这里，再判断是否值得沉淀。

## 使用方式

新输入包括：截图、链接、视频课片段、电子书摘录、客户问题、灵感、聊天记录、爆款选题。

默认流程：

```text
00_待处理
→ 判断价值
→ 进入对应系统目录
→ 移到 01_已入库 / 02_暂存观察 / 03_丢弃记录
```

## 判断标准

### 高价值

- 能反复调用；
- 能生成方法卡；
- 能成为内容选题；
- 能支撑商业判断；
- 能进入 ZenoAIHome 或 OPC 项目；
- 能成为故事/案例/表达素材。

### 暂存

- 有启发但用途不清；
- 需要更多案例验证；
- 可能属于未来项目。

### 丢弃

- 只有热闹没有结构；
- 低质量观点；
- 和长期目标无关；
- 会诱导低质模板化内容。
