# Skill 与 Agent 协作

## 工具分工

- Hermes：对话、判断、调度、写入本地资产。
- Codex：读写项目文件、批量整理、结构调整、代码/文本执行。
- dbskill：商业诊断、聊天室、问题拆解、内容/标题/开头辅助。
- Obsidian：本地资产库和导航系统。

## 协作边界

Agent 负责把资料变成更好的起点，不负责替 Zeno 完成所有表达。

## 常用模式

### 1. 输入入库

```text
资料 → 价值判断 → 灵感卡/方法卡/项目卡 → 索引更新
```

### 2. 爆款选题处理

```text
爆款选题 → 检索旧观点 → 素材膨胀 → 旧观点回避 → 人工写稿素材包
```

### 3. 项目推进

```text
想法 → 商业判断 → 最小测试 → 产品假设 → 复盘
```

## 禁止

- 不默认批量生成最终稿。
- 不绕过 Zeno 的审美判断。
- 不用 Agent 语言污染面向读者的内容。
