# Zeno OS 方法库索引

> 这里是可调用方法的入口。原始文件大多仍保留在原目录，本目录先做复制索引与升级候选，避免破坏旧链接。

## 使用原则

- 方法用于检索、判断、膨胀、反驳、拆解和组织，不用于一键代写重要逐字稿。
- 同一问题尽量调用多个方法，不依赖单一模板。
- 方法卡后续可逐步升级成 Hermes Skill 或 dbskill 子流程。
