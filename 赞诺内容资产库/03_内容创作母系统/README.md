# 内容创作母系统

> 这里不是 AI 代写区，而是 Zeno 的内容燃料、旧观点、选题方法和人工写稿起点。

## 目录

- `00_旧观点库/`：从历史文稿中抽出的观点、判断、常用说法。
- `01_爆款选题库/`：只记录选题，不抄内容，不看正文路径；可放内容雷达来源库、战报模板和高潜选题记录。
- `02_素材膨胀记录/`：围绕某个选题的 dbskill/chatroom 扩张记录。
- `03_人工写稿素材包/`：给 Zeno 人工写稿前使用的素材包。
- `04_标题与开头库/`：标题、开头、钩子，只做局部辅助。
- `05_平台表达规则/`：公众号、小红书、短视频、网站的表达边界。
- `06_内容复盘/`：发布反馈、数据、读者反应、下一轮选题。

## 原则

1. 爆款选题只借题，不借内容。
2. 先检索自己的旧观点，再决定复用或回避。
3. 素材膨胀不是 AI 写稿，而是扩大可写材料。
4. 最终逐字稿由人完成，AI 只做辅助。
5. 同一选题可以多次开发，但每次要有不同角度。
