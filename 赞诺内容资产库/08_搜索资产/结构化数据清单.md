## 优先实施清单

**第一优先：文章页 FAQPage Schema**

每篇文章的 FAQ 区要加上 FAQPage Schema，让 Google/Bing 可能在搜索结果里显示问答片段，同时提升 AI 引用概率。

代码示例：
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "装修预算为什么总是超支？",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "装修超预算最根本的原因是报价单结构性漏项..."
      }
    }
  ]
}
```

**第二优先：首页 Person Schema**

在首页加入个人信息的结构化数据，帮助建立 Knowledge Graph。

**第三优先：LocalBusiness Schema（服务页）**

南宁本地服务，加 LocalBusiness 有助于本地搜索。
