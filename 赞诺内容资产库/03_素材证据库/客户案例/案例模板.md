# 案例模板

> 用途：记录真实客户案例（脱敏后使用）

---

## 使用规则

1. 所有案例必须是真实发生的
2. 客户个人信息必须脱敏（不用真名、不用具体地址）
3. 数据可以模糊处理（如「12-15万区间」而不是「12.3万」）
4. 编造的案例绝对不用，哪怕只是细节

---

## 案例记录格式

```
## 案例ID：[自动编号，如 2026-A-001]

**项目基本信息：**
- 房屋类型：住宅/商业/旧改/改造
- 面积：XX 平米左右
- 项目阶段：规划期/施工期/竣工后
- 时间：XXXX年XX月

**客户背景（脱敏）：**
- 购房/装修原因：
- 预算区间：
- 主要顾虑：

**核心问题：**
（发生了什么问题？）

**我的处理方式：**
（我是怎么解决的？）

**结果：**
（最终结果是什么？）

**可以用于哪些选题：**
-
-

**使用状态：** 未使用 / 已用于《文章标题》
```
