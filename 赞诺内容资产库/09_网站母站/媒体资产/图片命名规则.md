# 图片命名规则

> 图片只要进网站，就必须可读、可搜、可维护。

## 一、基本规则
- 小写英文或拼音
- 单词之间用 `-`
- 不用空格、不用中文、不用下划线
- 尽量体现用途

## 二、推荐命名

### 文章封面
```text
cover.webp
```

### 文章配图
```text
scene-01.webp
scene-02.webp
scene-03.webp
```

### 资料预览图
```text
quote-checklist-preview.webp
budget-template-preview.webp
acceptance-checkpoints-preview.webp
```

### 品牌图
```text
avatar.webp
avatar-square.webp
logo.svg
```

## 三、目录结构

### 知识库备份
```text
09_网站母站/媒体资产/
  图片原图/
  封面图/
  文章配图/
  资料预览图/
  品牌头像/
```

### 网站公开路径
```text
zeno-site/public/images/
  articles/
  resources/
  brand/
  topics/
  videos/
```

## 四、上传前检查
- 是否压缩
- 是否裁剪到合理尺寸
- 是否已经改成英文/拼音文件名
- 是否会暴露隐私
- 是否真的需要进网站

## 五、默认边界
- 原图留在知识库
- 网站只用最终版图片
- 视频不直接放大文件到仓库
