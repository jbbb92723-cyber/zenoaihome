## 模块二：我从哪里来

```
我最早的视角来自工地。

在工地上，你很快会明白：真正决定结果的，往往不是口号，
而是细节、责任和协作。跑工地、盯节点、对图纸、
跟客户解释预算为什么会超——这些年积累的，
不只是施工经验，更是一套判断现实的方式。

后来我发现，这套判断方式不只适用于装修，
也适用于生活和工作里的大多数问题。
一个装修现场，浓缩了你能在任何复杂关系里遇到的
大多数困难：信息不对齐、责任推诿、
短期利益与长期质量的冲突。

再后来，我开始认真学 AI，不是为了追风口，
而是为了把分散的经验组织成系统，
把重复劳动交给工具，把时间还给判断和创造。
```

## 模块四：从审美到工地，再到 AI

```
我做装修，也一直关心审美。

不是那种「追风格、贴样板」的审美，
而是空间如何服务人的生活——
动线顺不顺，光线让人松不松，
收纳符不符合真实使用频率。

在工地里，我开始理解：审美不是装饰问题，
是价值观问题。你真正在乎什么、
愿意为什么付出代价，最终都会显现在
一个空间的取舍逻辑里。

再往后是 AI。
我学 AI 不是为了变成技术人，
而是因为它让我能把经验沉淀得更系统。
以前一次客户沟通，记录散在微信、语音、纸笔和脑子里；
现在我会把沟通整理成结构化笔记，
用 AI 协助做版本对比、风险清单和后续动作。

经历的顺序是：做装修 → 看见审美 → 走进工地 → 遇见 AI。
但它们在我这里从来不是分开的，而是同一个问题的不同面向：
**在真实的约束里，如何做出更好的选择。**
```

## 模块五：我为什么不只讲装修

```
很多人以为我只会写装修。
但我真正关心的是更大的问题：
一个人如何在现实压力、审美追求、效率工具和长期目标之间，
做出不后悔的选择。

所以这里会出现这些主题：
— 装修与居住：回到真实生活，而不是图片幻觉。
— 美学与秩序：让空间和日常变得更舒展。
— 人性与判断：识别情绪、偏见和短期冲动。
— AI 与工作方式：把重复工作交给系统，把时间留给创造。
— 长期主义：在喧闹里保持节奏，慢慢变强。

这些主题不是强行拼凑，它们是我在工地、生活、写作和学习里，
反复遇到的同一类问题的不同面向。
```

## 模块六：我相信什么

- 真实比包装重要。
- 清晰比热闹重要。
- 过程比口号重要。
- 长期积累比短期爆发更可靠。
- 一线经验是最难被替代的认知来源。

## 模块七：我反对什么

- 不做传统装修公司官网式表达。
- 不做「焦虑驱动」的营销话术。
- 不做卖课式承诺和速成神话。
- 不用「家人们」「闭眼入」「全网最全」等流量话语。
- 不把读者当小白或流量，而是当有判断力的成年人。

---

## 模块八：我的内容边界

**我会写的**：
- 有真实经历支撑的判断
- 有清晰立场的观点
- 有来源可追的数据（或标注「待验证」）
- 有助于读者做出更好决策的内容

**我不写的**：
- 我没有实践过的方法
- 无法核实的二手案例
- 只为流量服务的热点跟风
- 堆砌关键词的 SEO 灌水文章
