# 项目库｜ZenoAIHome

## 项目定位

ZenoAIHome 帮助准备装修签约的业主，在签字前看懂报价、合同、增项、付款节点、材料品牌型号和工期交付风险。

## 项目边界

这里处理装修垂直项目，不承载整个 Zeno OS。

适合进入这里的资料：

- 装修报价案例；
- 合同条款风险；
- 客户咨询问题；
- 装修平台评论痛点；
- 网站文章和服务页；
- 报价审核服务 SOP；
- 签约前咨询产品。

不适合直接进入这里的资料：

- 全域流量方法；
- 一人公司方法论；
- AI 培训课程设计；
- 个人认知和读书笔记；
- 与装修无关的内容创作方法。

这些应进入 Zeno OS 上层目录或 OPC 项目。
