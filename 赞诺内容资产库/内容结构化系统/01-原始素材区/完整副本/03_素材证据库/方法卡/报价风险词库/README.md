# 报价风险词库 README

## 这个库是什么

这里不是普通选题库，也不是装修术语词典。

这里是 ZenoAIHome 的报价风险判断资产库。

每一个风险词，都对应业主在签约前容易误判的一个位置：

- 报价怎么写
- 哪里可能模糊
- 后期怎么变成增项
- 签约前该问什么
- 怎样写进报价单或合同更安全
- 如何转成小红书、公众号、网站、服务检查项

## 为什么要建这个库

ZenoAIHome 的核心不是做“装修知识大全”。

核心是：

> 帮业主在签约前看懂装修报价风险。

小红书风险词系列只是外层内容。  
真正要沉淀的是背后的风险标签和判断方法。

当风险词持续积累后，它可以变成：

- 小红书图文系列
- 公众号深度文章
- 网站 SEO/GEO 内容
- 报价审核工具字段
- 人工精审检查清单
- 咨询问题库
- 合同联审提示词
- 业主签约前自查资料包

## 使用原则

1. 一篇只讲一个风险词。
2. 不制造焦虑，但要指出真实边界。
3. 不讲装修百科，只讲签约前判断。
4. 每个风险词都要能转成服务检查项。
5. 小红书讲“可收藏”，公众号讲“为什么”，网站讲“可搜索可引用”，服务讲“怎么帮你看”。

- 更安全的写法是什么？
- 可以转成哪些内容？
- 可以进入哪个服务字段？
