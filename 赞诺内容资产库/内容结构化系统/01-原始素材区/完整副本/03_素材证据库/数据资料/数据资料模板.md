# 数据资料模板

> 用途：收录可以在文章中引用的数据、统计、研究报告

---

## 使用规则

1. 所有数据必须有明确来源
2. 标注数据的时效性（发布年份）
3. 没有来源的数据不记录，不使用
4. 官方/行业机构数据优先

---

## 数据资料记录格式

```
## 数据ID：[日期+序号，如 2026-D-001]

**数据内容：**
（核心数字是什么）

**原文引用：**
（尽量保留原文表述）

**来源：**
- 机构名称：
- 报告/文章名称：
- 发布时间：
- 链接（如有）：

**数据时效：** 
（这个数据到什么时候可能过时）

**适用选题：**
-
-

**使用状态：** 未使用 / 已用于《文章标题》
```
