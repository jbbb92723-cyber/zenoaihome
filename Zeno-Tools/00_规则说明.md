# Zeno-Tools 规则说明

## 定位

这里存放脚本、自动化规则、Agent 协作配置、软件配置和本地工具链相关文件。

## 可以放这里

- 自动化脚本、批处理、数据同步工具、发布辅助工具。
- Agent 使用说明、技能规则、协作规则、工作流配置。
- 编辑器、浏览器、QA、部署、监控等工具相关配置。
- 临时工具项目、实验性工具脚本、非网站运行必需的辅助程序。

## 不放这里

- 正式网站前端、后台、API 的核心源码。
- 文章、素材、笔记、运营资料。
- 运行缓存、崩溃日志、浏览器 profile、构建产物。
- 密钥、账号、私有 token、`.env.local`。

## 归类原则

如果一个文件主要回答“如何辅助生产、检查、发布、自动化处理”，优先归入这里。

如果这个工具已经成为正式网站运行链路的一部分，例如线上 API 必需脚本，应先放入 `待归类文件夹`，由人工确认归属。
