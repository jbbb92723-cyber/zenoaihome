# Zeno-Tools 规则说明

## 定位

脚本、工具链和辅助配置。不是 Skill 目录——Skill 在 `.hermes/skills/` 里。

## 目录结构

```
Zeno-Tools/
├── 00_规则说明.md              ← 你正在读的文件
├── scripts/
│   └── publish.py              ← CTA检查脚本（日常用：发布前跑一次）
├── batch-generators/           ← 一次性批量生成脚本（跑完归档，不常用）
│   ├── generate_zeno_batch*.py  （12个内容批量生成脚本）
│   ├── ocr_*.py                 （PDF OCR 提取工具）
│   └── sync_*.py                （Skill 同步脚本）
└── taste-skill-main/           ← 设计品味工具（可迁移到 .hermes/skills/）
```

## 可以放这里

- 自动化脚本、批处理、发布辅助工具
- 一次性生成脚本（跑完归档到 `batch-generators/`）
- 编辑器、QA 等工具相关配置

## 不放这里

- **Hermes Skills** → 放 `G:/Hermes智能体/.hermes/skills/`
- **人物蒸馏 Skill** → 放 `Zeno-Content/perspective-skills/`
- 文章、素材、笔记 → 放 `Zeno-Content/赞诺内容资产库/`
- 网站源码 → 放 `Zeno-Code/zeno-site/`
- 密钥、token、`.env.local`
