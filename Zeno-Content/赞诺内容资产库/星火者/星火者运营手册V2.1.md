# 卷一：共同体定义

## 1.1 星火者是什么

> **{已运行}**

星火者是由赞诺发起的成员制协作共同体，面向一人公司经营者、AI 实战者和正在转型的传统行业经营者。

它不是聊天群，也不是承诺收益的项目平台。成员在这里：

- 分享正在做的事
- 学习可复用的方法
- 认识能一起做事的人
- 在规则清楚的前提下自愿合作

"星火"是共同体的精神意象，"星火者"是成员的身份称谓。"赞诺·星火者"是正式对外品牌名称。品牌辨识度依靠体系化内容输出、真实案例积累和持续运营建立，不依赖名称独占性。商标注册和平台名称保护需另行处理，不在本手册范围内。

## 1.2 产品四层结构

> **{已运行}**

| 层级 | 定位 | 性质 | 对外承诺 |
|------|------|------|----------|
| **星火者共同体** | 核心产品，成员制协作网络 | 付费成员制 | 连接、分享、学习、活动、合作机会 |
| **火种技能读书会** | 学习与内容机制 | 共同体内部活动 | 固定主题、真实案例、成员交流 |
| **共燃项目** | 成员自愿协作机制 | 项目制合作 | 规则透明，不承诺收益 |
| **企业 AI 服务** | 广西赞诺数智科技有限公司商业服务 | 独立商业合同 | 单独签约、报价、结算 |

**重要区分**：

- 加入星火者共同体 ≠ 获得企业服务收入
- 参与共燃项目 ≠ 保证项目成功或有收入
- 星火值 ≠ 货币、股权、债权或收益承诺
- 企业服务由赞诺独立签约，成员参与交付时另行签署项目合作协议

## 1.3 暂不启动的内容

> **{暂不启动}**

以下内容在首期试运行期间不启动，待真实项目、合同和财务机制成熟后再单独评估：

- 投资孵化
- 分红池
- 平台收益分配
- "加入就能一起赚钱"类暗示

# 卷二：成员体系

## 2.1 适合与不适合

> **{已运行}**

**适合加入的**：
- 正在经营一人公司的人
- 有明确技能和正在推进项目的人
- 把传统行业经验转化为 AI 工作流的人
- 愿意分享自己正在做的事的人
- 愿意与其他人合作的人

**不适合加入的**：
- 只想领取资源、不做任何贡献的人
- 只想找客户、发广告的人
- 把共同体理解为就业或投资机会的人
- 期待平台保证收入的人
- 不愿意遵守公约的人

## 2.2 成员等级

> **{试运行}**

取消按星火值自动晋升。改为按实际参与和贡献评价：

| 等级 | 条件 | 权益 |
|------|------|------|
| **星火者** | 通过申请 → 完成点火 → 参与至少一次活动 | 加入主群、参与读书会、查看内容沉淀 |
| **核心星火者** | 连续 3 个月有有效参与 + 完成至少一次分享或协作 | 优先认领项目、核心讨论、活动优先 |
| **燎原合伙人** | 独立发起并完成至少一个项目闭环 | 发起项目、代表共同体对外连接、参与运营决策讨论 |

星火值是参考记录，不是自动晋升的充分条件。

## 2.3 准入流程

> **{试运行}**

**完整流程**：

1. 提交申请表（含基本信息、技能标签、当前在做什么、为什么想加入）
2. 15 分钟交流（主理人或运营负责人，了解真实意图和能力匹配度）
3. 明确告知适合与否（不适合的对象直接告知，不勉强加入）
4. 通过后支付首期费用
5. 24 小时内完成"点火"（群内自我介绍 + 至少参与一个话题）
6. 7 天内完成至少一次有效互动（参与读书会讨论、发起一个提问、或分享一次经验）

## 2.4 退出机制

> **{已运行}**

- **主动退出**：成员可随时退出，费用按当期招募说明处理
- **沉默退出**：连续 3 个月无有效互动，运营提醒。提醒后 1 个月仍无互动，自动退出
- **违规退出**：违反公约的行为，一次提醒，二次退出
- **退款**：加入 7 天内可无条件退款。7 天后按当期招募说明执行

# 卷四：星火值系统

## 4.1 性质

> **{试运行}**

> **星火值是共同体内部的贡献记录，不是货币、股权、债权，也不代表任何收益承诺。**

星火值用于记录成员在共同体内的活跃度和贡献，作为荣誉参考和部分活动优先权的依据。

## 4.2 获取方式

| 行为 | 星火值 | 说明 |
|------|--------|------|
| 完成点火（群内自我介绍） | +5 | 一次性 |
| 参与火种技能读书会 | +3/次 | 需有实质发言 |
| 主动分享经验/复盘 | +5~15 | 视内容质量 |
| 发起一个有意义的讨论 | +3 | 引发 5 人以上参与 |
| 参与共燃项目 | +10/项 | 需实际交付 |
| 独立发起并完成项目 | +30/项 | 视完成质量 |
| 引荐合适成员加入 | +5/人 | 引荐人通过准入流程 |
| 帮助其他成员解决问题 | +2~10 | 视帮助效果 |

## 4.3 用途

- 活动优先报名
- 项目优先认领
- 内容展示（成员排行榜/荣誉墙）
- 核心讨论参与
- 年度荣誉（年度星火者）

## 4.4 不使用星火值的事项

以下事项不以星火值为依据：

- 分红（没有分红制度）
- 收益分配（项目收益按项目合同结算）
- 永久免费资格（没有永久免费制度）
- 自动晋升（晋升由运营评估，星火值只作参考）

# 卷六：项目协作（共燃项目）

## 6.1 基本原则

> **{试运行}**

- 共燃项目是成员之间自愿发起的协作，共同体提供连接和规则框架
- 共同体**不保证**项目成功，**不保证**参与者有收入
- 涉及企业客户的项目，由广西赞诺数智科技有限公司或指定项目主体与客户单独签署服务合同
- 成员参与交付时，另行签署项目合作和结算确认文件

## 6.2 项目启动前必须明确的事项

每个共燃项目在启动前，必须在项目群内写清以下内容并经所有参与者确认：

| 事项 | 说明 |
|------|------|
| 客户是谁 | 内部项目注明"内部" |
| 谁签合同 | 签约主体 |
| 谁收款和开票 | 财务负责方 |
| 项目总价 | 合同金额 |
| 成本构成 | 直接成本明细 |
| 分配方式 | 按总价还是净利润分配 |
| 每人负责什么 | 角色和交付物 |
| 交付验收标准 | 什么时候算"做完了" |
| 延期/退款/违约 | 出现意外怎么处理 |
| 知识产权归属 | 交付物归谁 |
| 保密义务 | 什么不能对外说 |
| 平台管理费 | 如有，以项目合同为准 |

## 6.3 平台管理费

平台管理费不在本手册中以固定比例承诺。具体费用以每个项目的合作确认文件为准。

试运行期间，可以先试行一个固定比例（如 10%~15%）作为运营参考，但必须：
- 在项目合同中单独写明
- 获得所有参与者签字确认
- 写明平台该笔费用对应的服务内容（项目撮合、合同模板、工具支持、复盘组织等）

# 附录 B：火种技能读书会 SOP

> **{试运行}**

**周期**：每两周一次
**时长**：60-90 分钟
**形式**：线上（腾讯会议/飞书会议）

**流程**：

1. **选题**（会前一周）：运营助理在群内征集话题 + 主理人确定主题
2. **通知**（会前 3 天）：发布正式通知，含主题、时间、主持人
3. **案例准备**：主持人准备 1~2 个真实案例
4. **会议流程**：
   - 主持人开场（5 分钟）：主题引入
   - 案例拆解（20 分钟）：讲清楚一个真实案例的获客-转化-交付链条
   - 自由讨论（30 分钟）：成员提问、分享类似经验
   - 总结（5 分钟）：提炼可复用的方法

# 附录 D：FAQ

> **{试运行，持续更新}**

**Q：星火者是什么？**
A：是由赞诺发起的成员制协作共同体。不是培训、不是投资平台、不是就业机会。

**Q：加入有什么好处？**
A：认识同行、学习方法、参与项目、获得支持。但所有收益都来自于你自己的行动。

**Q：星火值能换钱吗？**
A：不能。星火值是贡献记录，不是货币。

**Q：能保证赚钱吗？**
A：不能。星火者提供方法和机会，不保证任何人的收入。

**Q：平台抽成多少？**
A：每个项目单独约定，不在手册中锁定固定比例。

**Q：能退款吗？**
A：加入 7 天内无条件全额退款。之后按当期招募说明。
