# 星火者 · 共同体说明书

> 一人是星火，众人是燎原。

---

## 我们是谁

星火者是由赞诺发起的成员制协作共同体，面向一人公司、AI实战者和正在转型的传统行业经营者。

它不是聊天群，也不是承诺收益的项目平台。成员在这里分享正在做的事，学习可复用的方法，认识合适的同行，并在规则清楚的前提下自愿合作。

说"我是星火者"的人，是在说：我是独立的，但不是孤立的。我有自己的事，但不拒绝跟对的人一起做更大的事。

---

## 名字的含义

**星火者 = 天·地·人**

- **星**（天）— 方向、视野、不迷失
- **火**（地）— 能量、燃烧、传递
- **者**（人）— 行动、身份、承诺

> "星火"是共同体的精神意象，"星火者"是成员身份称谓。品牌辨识度主要依靠"赞诺·星火者"、固定话语体系、真实案例和持续运营建立。

# 星火者 · 成员公约

> 这份公约是每个星火者加入时确认的规则。不是"用户协议"，是"我们约定好的事"。

---

## 一、你的权利

### 1.1 内容权利

- 免费访问知识星球所有精华帖、猎场档案、星火档案
- 参加火种技能读书会
- 获取每周火种速递摘要

### 1.2 协作权利

- 报名参与共燃项目（猎场）
- 达到条件后发起共燃项目
- 在规则透明的前提下与成员自愿合作

### 1.3 活动权利

- 参加线上线下活动
- 申请在火种分享会发言

### 1.4 退出权利

- 72小时内无条件全额退款
- 随时申请暂熄（保留知识星球和星火值，退出微信群）
- 随时恢复参与

---

## 二、你的义务

### 2.1 参与义务

- 加入后24小时内完成点火（自我介绍）
- 加入后7天内完成至少一次有效互动
- 遵守所在群的参与节奏

### 2.2 诚信义务

- 分享内容必须真实——真人、真事、真数据
- 共燃项目中承诺的交付必须完成；因客观原因无法完成的，及时沟通并退出

### 2.3 尊重义务

- 尊重其他成员的隐私、时间和边界
- 不在群内发布无关广告
- 不将群内信息（成员联系方式、项目细节）泄露给外部

---

## 三、隐私保护

- 成员的自我介绍和分享内容，仅在共同体内部分享
- 未经本人明确同意，不对共同体外部公开成员个人信息
- 精华帖经作者审核后方可对外（公众号、网站）转载
- 知识星球内容对全体星火者开放，无二次付费墙

---

## 四、内容授权

- 成员在知识星球发布的原创内容，著作权归作者所有
- 成员授权赞诺·星火者在共同体内部分享和使用该内容（精华帖整理、星火档案）
- 如需对外发布（公众号、网站、播客），需经作者另行确认

---

## 五、退出机制

| 情况 | 处理方式 | 恢复条件 |

## 三、火种分享会 🟡试运行

**频率**：每周一次
**形式**：腾讯会议 / 微信群语音 + 文字
**轮值规则**：由上一周的分享者担任下周主持（形成接力）

**分享内容标准：**
- 必须包含：我做了什么、结果如何（数据）、踩了什么坑
- 不要求成功——失败复盘比成功案例更珍贵
- 分享后7天内，分享内容整理发布到知识星球

**参会纪律：**
- 不要求全部到场
- 来了就要发言——"我有启发因为XX"也行

### 火种分享会大纲模板

```
🎤 火种分享：[主题]
【结果先行】先告诉大家我做到了什么（数据）
【项目背景】为什么开始做这个
【关键步骤】3-5步，每步有动作有结果
【最大的坑】踩过最惨的一次
【如果再做一次】会怎么改
【你可以试试】给听众的一个可操作的建议
```

# 星火者 · 项目合作规则

> 这份文档规定共燃项目的协作规则。每个项目启动前，所有参与者确认本规则。

---

## 一、项目类型

| 类型 | 代号 | 频率 | 时长 | 人数 |
|------|------|------|------|------|
| 月度探索 | **共燃·猎场** | 视发起情况 | 30天 | 3-8人 |
| 季度深度 | **共燃·炉边** | 每季1次 🔵计划中 | 2天线下 | 10-20人 |
| 年度大型 | **共燃·燎原** | 每年1次 🔵计划中 | 3个月 | 不限 |

> 首期试运行只启动猎场。炉边和燎原为计划中，待核心成员数量达到阈值后启动。

---

## 二、发起规则

### 2.1 发起条件

- 发起人（点火人）需为核心星火者或主理人特批
- 在成员主群发起

### 2.2 发起必须说清五件事

1. **做什么**：项目目标（一句话能说清）
2. **需要谁**：角色 + 技能要求（至少列出2个角色）
3. **分润方案**：收益分配比例（必须在组队前公布）
4. **时间线**：启动、里程碑、结项日期
5. **交付标准**：什么算成功，什么算失败

### 2.3 发起模板

```
🔥 猎场招募：第00X号

【项目名称】一句话说清楚
【点火人】[名字]
【一句话目标】30天内要拿到什么结果
【项目描述】3-5句话说明白要做什么
【所需角色】
  · [角色A]：[技能要求] | 1人
  · [角色B]：[技能要求] | 1-2人
【分润方案】总收益分配比例（组队前公布，不可后改）
【时间线】
  启动：X月X日
  Kickoff会议：X月X日（组队后48h内）
  结项交付：X月X日
【报名方式】评论区报名 + 一句话介绍相关经验
【截止时间】X月X日
```

## 三、合同与主体

### 3.1 签约主体

涉及企业客户的项目，由**广西赞诺数智科技有限公司**或指定项目主体与客户单独签署服务合同。

### 3.2 成员参与

成员参与交付时，另行签署项目合作确认文件，明确以下内容：

| 事项 | 必须写清 |
|------|----------|
| 客户是谁 | 甲方全称 |
| 谁签合同 | 签约主体 |
| 谁收款和开票 | 收款账户、开票方 |
| 项目总价 | 合同总金额 |
| 成本如何计算 | 哪些算成本、谁承担 |
| 收益分配 | 收益还是净利润参与分配、具体比例 |
| 分工与交付 | 每人负责什么、验收标准 |
| 时间节点 | 每个节点的交付物和截止日期 |
| 延期处理 | 延期如何通知、如何处理 |
| 退款和违约 | 客户退款如何处理、成员违约如何处理 |
| 知识产权 | 交付物归属、成员可否复用 |
| 保密义务 | 客户信息的保密范围和期限 |
| 平台是否承担交付责任 | 明确！ |

### 3.3 平台管理费

> 涉及企业客户的项目，具体服务费、成本、结算方式和平台管理费，以项目合同为准，不以本手册中的比例作为自动承诺。

## 四、参与规则

### 4.1 报名与组队

- 自由报名，点火人拥有最终选择权
- 组队后48小时内，在临时项目群开kickoff会议
- 团队人数上限8人

### 4.2 中途退出

- 不扣星火值
- 需及时通知点火人和团队
- 退出者的未完成工作由团队重新分配
- 退出记录影响社区信用

### 4.3 进度跟踪

- 每周在项目群发布进度简报（做了什么/卡在哪/下周计划）
- 连续2周无进度更新 → 主理人介入

## 六、知识产权

- 项目交付物的知识产权归属以项目合作确认文件为准
- 如无特别约定，交付物著作权归客户（甲方）
- 成员个人在项目中贡献的方法、流程、经验，成员个人保留复用权
- 项目复盘内容经脱敏后，星火者共同体可在内部使用

## 七、争议处理

### 7.1 一般争议

成员间争议 → 先由当事双方直接沟通 → 无法解决的，主理人介入调解。

### 7.2 与客户争议

由签约主体（广西赞诺数智科技有限公司或指定项目主体）与客户按合同约定处理。成员配合提供相关交付记录。

### 7.3 分润争议

- 分润方案在组队前公布，组队后不可更改
- 如有争议，以组队前公布的分润方案为准
- 实际分配以客户实际付款金额为基础

- 项目收益取决于市场、客户、执行等多种因素，星火者平台不保证项目成功或盈利
- 成员参与项目应自行评估风险和自身能力
- 平台不对项目结果承担交付责任或连带责任（以项目合同和项目合作确认文件为准）
