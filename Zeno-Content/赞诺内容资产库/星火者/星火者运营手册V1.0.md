## 卷首 · 我们的定义

**我们是谁**

星火者。AI时代，一人公司的分布式协作网络。

**我们的宪章**

> 一人是星火，众人是燎原。

这句话是星火者的最高准则。它定义了我们之间的关系——不是雇佣，不是师生，不是买卖。是独立个体之间的彼此点燃、彼此成就。

- 每一人公司，都是一束值得被看见的光。
- 聚在一起，不是为了取暖，是为了烧得更旺。

### 2.2 各群定位

**新手群（当前约100人）**

- 定位：广撒网，筛选期。所有人付费后第一站。
- 核心动作：点火仪式（24小时内自我介绍）。
- 筛选节奏：每2周一次升群——在新手群活跃、分享、被看见的人，邀请进入高手群。
- 清理节奏：每月清理一次。从未发言、从未互动的，友善请出新手群，保留知识星球访问权（暂熄状态）。
- 不做什么：不在新手群深度讨论。新手群是"看见别人在做什么"的地方，不是深度交流的地方。深度留在高手群。

**高手群（当前约100人）**

- 定位：深度连接，精华产出。这是星火者的发动机。
- 氛围：小圈子、高信任、敢说真话。人数控制在150以内。
- 核心动作：每周一次轮值分享（"火种分享会"），所有人轮流，没有例外。
- 猎场发起：所有星火计划的组局只在这里发起——让新手看到"进高手群才有真东西"。
- 不做什么：不闲聊灌水。高手群的信息密度是筛选的延伸——进来的人会觉得"这才是我想待的地方"。

### 2.3 知识星球

| 维度 | 做法 |
|------|------|
| 定价 | 与门票同步——买门票=进知识星球+微信群，一体 |
| 精华帖 | 每一篇标注：谁、做了什么、赚了多少/省了多少、花了多长时间 |
| 新人onboarding | 进来先刷3天精华帖，再进微信群。看完再聊，认知同频 |
| 星火值公示 | 每月1号在知识星球发布星火值排行榜 |
| 星火档案 | 每次猎场计划的交付物=一篇精华帖。一年后你有50篇精华=一本书的底稿 |

## 卷四 · 星火计划（项目协作）

### 4.1 三种模式

| 类型 | 频率 | 时长 | 人数 | 目标 |
|------|------|------|------|------|
| **星火·猎场** | 每月1次 | 30天 | 3-8人 | 探索新项目/新赛道，不熟的人一起做事 |
| **星火·炉边** | 每季1次 | 2天线下 | 10-20人 | 深度复盘+协作+关系加固 |
| **星火·燎原** | 每年1次 | 3个月 | 不限 | 跨成员大型协作，产出标杆案例 |

### 4.2 猎场规则

**发起（点火人）：**
1. 在高手群发起
2. 说清三件事：做什么、需要谁（角色+技能）、怎么分钱
3. 必须在30天内有交付物

**参与（共燃者）：**
1. 双向认领，自由组队
2. 中途退出不扣星火值，但下次组队会有信用记录
3. 交付失败（客观原因）→ 不扣分，写复盘；交付失败（主观跳票）→ -15

**交付物：**
- 成功 → 项目复盘（方法论+数据）
- 失败 → 失败复盘（试了什么、为什么不成、如果再来一次会怎么改）
- 无论成败，交付物上传知识星球，成为星火档案

### 4.3 猎场日历（V1.0首年）

```
2026年 Q3（8-9月）
  8月：第一个猎场（项目方向由高手群投票决定）
  9月：第二个猎场

2026年 Q4（10-12月）
  10月：第一个炉边（南宁线下）
  11月：第三个猎场
  12月：星火半年盘点 + 全员信

2027年 Q1-Q2
  3月：燎原计划启航
  6月：第一届星火大会
```

## 卷五 · 进入与退出

### 5.1 定价模型

| 阶段 | 价格 | 说明 |
|------|------|------|
| 首发期（前100人） | ¥1,999/年 | 当前价格，满100人上调 |
| 第二档（101-200人） | ¥2,499/年 | |
| 第三档（201人+） | ¥2,999/年 | 之后每期发行价调整 |
| 续费 | 当期价格 × 0.8 | 老成员永远是8折 |

**3天无条件退款。** 这是信任的底线。

### 5.2 点火仪式（新人加入）

新成员付费后，24小时内完成点火仪式——在知识星球或新手群发布自我介绍。

**格式：**
```
【我是谁】
【我的技能/擅长什么】
【正在做的项目】
【需要的资源/帮助】
【我能提供给其他星火者的价值】
```

点火成功后，老成员回复："点火成功，欢迎星火者！"

### 5.3 升群机制

新手群 → 高手群：每2周评估一次，由核心群推荐+主理人确认。
标准：在新手群活跃分享、主动引火、被多人@过的人。

### 5.4 退出机制

| 情况 | 处理 |
|------|------|
| 长期潜水（>1个月零互动） | 新手群→友善请出，保留知识星球（暂熄）；高手群→降级回新手群 |
| 破坏信任（跳票/广告/人身攻击） | 一票否决，清出所有群，不退费 |

**暂熄者**仍可通过知识星球访问内容。恢复活跃后可申请回归。

### 7.1 精华帖标准

每一篇星火精华帖，必须包含：

```
✅ 谁做的（星火者名字）
✅ 做了什么（项目/尝试一句话说清）
✅ 结果（赚了多少/省了多少/学到了什么，有数字就用数字）
```

**精华帖不是写论文，是写战报。** 八百字以内。有截图、有数据、有情绪。

| 线下聚会 | 一人写一句收获，拼成一篇文章 |

## 卷八 · 信任与底线

### 8.1 三条铁律

**1. 不骗人。** 精华帖必须是真人真事真钱。没有结果的分享，不能叫精华帖，叫心得体会。两者要分开标注。

**2. 分钱透明。** 猎场项目分润方案在组队前公布，结项后公开。任何分钱不爽快的人，离开星火。

**3. 信息流动。** 知识星球的精华帖对全体星火者开放。不搞"付费内容再付费"。门票之后，没有二道门。

### 8.2 禁止事项

- 发布无关广告、商业推广
- 讨论政治敏感话题
- 人身攻击、贬低、嘲讽
- 私下拉群批量添加好友进行商业推销
- 泄露社群内部非公开信息（猎场项目细节、其他星火者联系方式）

### 8.3 主理人的承诺

> 如果我自己加入星火者觉得不值，我不收这个钱。
> 如果星火者续费率低于60%，说明我有问题——不是在运营，是在收割。
