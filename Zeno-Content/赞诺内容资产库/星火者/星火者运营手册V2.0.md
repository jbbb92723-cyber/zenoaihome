## 卷首 · 我们的定义

### 我们是谁

星火者。AI时代，一人公司的分布式协作网络。

不是社群，不是联盟——"星火者"是一个身份。说"我是星火者"的人，是在说：我是独立的，但不是孤立的。我有自己的事，但不拒绝跟对的人一起做更大的事。

### 名字的含义

**星火者 = 天·地·人**

- **星**（天）— 方向、视野、不迷失。抬头看天，知道往哪走。
- **火**（地）— 能量、燃烧、传递。脚踏实地，手里有活，能点燃别人。
- **者**（人）— 行动、身份、承诺。不是围观者，是那个让星和火发生关系的人。

"星火"是一个现象，撞名是必然的——好意象人人想用。"星火者"不是现象，是身份。没人能跟你撞"星火者"，因为这个名字定义的不是"这个社群叫什么"，而是"加入这个社群的人是谁"。

### 对外自称分场景

| 场景 | 自称 | 为什么 |
|------|------|--------|
| 内部互称 | "我是星火者" | 五个字，身份一步到位 |
| 冷启动 | "我们在找星火者" | 不说"社群招商"，说"找人" |
| 正式活动 | 星火者——AI一人公司分布式协作网络 | 定位句，一次说清楚 |
| 政府对接 | 赞诺·星火者AI产业服务 | 母品牌压舱，"产业服务"锚定交付 |
| 企业商务 | 星火者·企业AI赋能 | 直接说能帮你做什么 |
| 公司签约 | 广西赞诺数智科技有限公司 | 法律实体，开票用 |
| 知识星球 | 星火者（付费社群） | 平台内自然搜索 |

### 我们的宪章

> 一人是星火，众人是燎原。

这句话是星火者的最高准则。它定义了三种关系：

- **你和自己的关系**：你是独立的个体。一人公司，自带光芒，有自己的轨道。
- **你和其他星火者的关系**：不是雇佣，不是师生，不是买卖。是彼此点燃，彼此成就。
- **你和世界的关系**：聚在一起不是为了取暖，是为了烧得更旺。从广西出发，点亮全国。

### 我们的信念

- 每一人公司，都是一束值得被看见的光。
- 聚在一起，不是为了取暖，是为了烧得更旺。
- 星星之火，可以燎原。从广西出发，点亮全国。

### 2.2 新手群运营细则

**定位**：所有付费成员的第一站。广撒网，筛选期，不是终点站。

**当前规模**：约100人。上限不设，但保持流动——每月有人升到高手群，有人被清理。

**核心KPI**：点火仪式完成率 > 80%（付费后24小时内自我介绍）。

**新人onboarding流程（付费后24小时）：**

```
Step 1: 付费完成 → 自动发送欢迎消息（模板见附录D）
         包含：知识星球链接 + 新手群二维码 + 点火仪式说明

Step 2: 进入新手群 → 群公告自动弹出（含群规 + 点火模板）
         老成员无主动欢迎——等新人自己点火

Step 3: 新人发布自我介绍（点火仪式）→ 格式见附录B
         老成员回复："点火成功，欢迎星火者！[名字]！"

Step 4: 点火后3天内 → 至少产出一条有价值内容
         （在知识星球发帖 / 在群内分享经验 / 参与话题讨论）

Step 5: 点火后1周 → 仍未发过言的，鱼丸私聊关心："兄弟，方便的话可以
         在星球里跟大家介绍一下自己，让大家认识你～"
```

**清理节奏**：

| 时间节点 | 动作 | 提前告知 |
|----------|------|----------|
| 每月25号 | 发布清理预告："本月底将清理潜水超过1个月的同学" | 提前3天 |
| 每月最后一天 | 执行清理：移除零互动成员 | 私聊通知：暂熄 |

**暂熄说明**：被清理≠被踢出。暂熄者保留知识星球访问权，可随时申请回归新手群。暂熄只是"你现在不在火堆旁边了，但火种还在"。

**禁止事项**：
- 不在新手群发起猎场计划（猎场在高手群发起）
- 不在新手群深度讨论商业细节（深度留在高手群）
- 不纵容灌水——新手群是"看到别人在做什么"的地方，不是闲聊群

### 2.3 高手群运营细则

**定位**：深度连接，精华产出，星火者的发动机。人数严格控制在150以内。

**进入条件（满足其一即可）**：
- 星火值 ≥ 100
- 在新手群连续3个月活跃（发言+分享+引火），由核心群推荐
- 主理人直接邀请（认可其潜力）

**核心运转机制：火种分享会**

```
频率：每周三晚上 20:00-21:30
主持：由上一周的分享者担任下周主持（形成接力）
形式：腾讯会议 / 微信群语音 + 文字

轮值规则：
- 每个高手群成员必须至少来一次（不做就是放弃高手群资格）
- 顺序由主理人排期，每月初公布当月排期
- 自己讲不了可以推荐外部嘉宾（"外火"），但推荐人需要做开场介绍

分享内容标准：
- 必须包含：我做了什么、结果如何（数据）、踩了什么坑
- 不要求成功——失败复盘比成功案例更珍贵
- 分享后7天内，分享内容整理发布到知识星球（精华帖候选）

参会纪律：
- 不要求全部到场，但至少50%的人在
- 来了就要发言——"我有启发因为XX"也行，拒绝纯潜水
```

**高手群退出/降级机制：**
- 连续3个月零发言 → 降级回新手群
- 被连续2次排到轮值分享但拒绝 → 降级回新手群
- 暂熄：主动声明"最近太忙，先回新手群" → 保留星火值，随时可申请回归

## 卷三 · 星火值系统

### 3.1 设计原则

星火值的核心设计参考了生财有术的龙珠体系，但有一个关键差异：

| | 龙珠（生财有术） | 星火值（星火者） |
|------|------|------|
| 获取方式 | 精华帖为主 | 分享、协作、引火、拉新——多维获取 |
| 兑换 | 门票、分红、闭门会 | 门票、进阶权、分红权、终身门票 |
| 可见性 | 不公开排名 | 每月公示TOP30 |
| 定价 | 市场价浮动（曾达上万元） | 固定兑换体系，不浮动 |

**核心原则：星火值不换现金。** 换现金 = 激励变交易。换权益 = 激励变归属。一个说"我有200星火值"的人，比说"我在这里赚了200块"的人忠诚度高十倍。

## 卷五 · 课程体系

### 5.1 课程设计哲学

星火者的课程不是"教你赚钱"，是"让你跟已经赚到钱的人一起做"。

核心原则：
- **不做录播课**。课程是活的——每次火种分享会就是一门课，每次猎场组队就是一堂实战。
- **不请外部讲师**。讲师全都是星火者自己。让已经在做的人来讲正在做的事。
- **课程 = 知识的组织方式**。星火者的内容散落在精华帖、火种分享、猎场复盘里。课程体系的作用是把这些散落的知识变成结构化的学习路径。

### 8.1 四条铁律

**1. 不骗人。** 精华帖必须是真人真事真钱。没有结果的分享，叫心得体会，不叫精华帖。两者要分开标注。

**2. 分钱透明。** 猎场项目分润方案在组队前公布，结项后公开。任何分钱不爽快的人，离开星火。不是"踢出"——是"星火者不跟分钱不爽快的人一起做项目"。分钱透明不需要惩罚机制，因为不透明的人自然没人跟他组队。

**3. 信息流动。** 知识星球的精华帖对全体星火者开放。门票之后，没有二道门。不搞"付费内容再付费"。

**4. 入口即出口。** 3天无条件退款。退款的不是你的敌人——是你的产品不够好。如果你发现退款率>20%，不是用户在收割你，是你的产品在收割用户。

### 8.2 禁止事项（完整版）

- ❌ 发布无关广告、商业推广
- ❌ 讨论政治敏感话题
- ❌ 人身攻击、贬低、嘲讽、阴阳怪气
- ❌ 私下拉群批量添加好友进行商业推销
- ❌ 泄露社群内部非公开信息（猎场项目细节、星火者联系方式等）

## 附录C · FAQ（常见问题标准回答）

**Q: 星火者到底是什么？**
A: 一群一人公司。互相分享怎么赚钱，一起组队做项目分钱。不是听课的群，是干活的地方。

**Q: 跟别的社群有什么不一样？**
A: 两个区别。第一，你不是"会员"，你是"星火者"——这是一个身份。第二，我们不搞大咖站台。每个人都要分享，每个人都要干活。潜水会被请出去。

**Q: 我什么都不会，能进吗？**
A: 你不会的，有人会。你会的，可以教别人。关键是——你得有一件你愿意做、正在做的事。哪怕刚开始。如果你只是想观望，这里不适合你。

**Q: 1999贵不贵？**
A: 看你跟什么比。跟一门录播课比，贵。跟你可能在这里找到的一个项目合伙人、一个信息差、一个帮你省了2万块的坑比——不贵。3天内觉得不值，全额退。

**Q: 为什么不叫社群/联盟？**
A: 群是松散的，联盟太正式。星火者是一个身份。说"我是星火者"，三个字就够了。

**Q: 续费率是什么？我为什么要关心？**
A: 你不需要关心。我需要。如果明年你不续费，说明今年你没得到你想要的。那是我的问题。
