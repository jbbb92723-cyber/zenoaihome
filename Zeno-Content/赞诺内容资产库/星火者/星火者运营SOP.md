## 二、成员主群运营

### 2.1 新人Onboarding（付费后24小时）

```
Step 1: 通过申请 → 支付 → 进入成员主群
        发送欢迎消息（模板见附录A）

Step 2: 进入群后 → 新人发布自我介绍（点火仪式）
        格式见附录B
        老成员回复："点火成功，欢迎星火者！[名字]！"

Step 3: 点火后3天内 → 至少产出一条有价值内容
        （在知识星球发帖 / 在群内分享经验 / 参与讨论）

Step 4: 点火后1周 → 仍未发过言的，助理私聊关心
```

### 2.2 清理节奏 🟡试运行

| 时间节点 | 动作 | 提前告知 |
|----------|------|----------|
| 每月25号 | 发布清理预告 | 提前3天 |
| 每月最后一天 | 执行清理：移除零互动成员 | 私聊通知：暂熄 |

**暂熄说明**：被清理≠被踢出。暂熄者保留知识星球访问权，可随时申请回归。暂熄只是"你现在不在火堆旁边了，但火种还在"。

### 2.3 禁止事项

- 不发布无关广告
- 不纵容灌水
- 不讨论政治敏感话题

## 三、火种分享会 🟡试运行

**频率**：每周一次
**形式**：腾讯会议 / 微信群语音 + 文字
**轮值规则**：由上一周的分享者担任下周主持（形成接力）

**分享内容标准：**
- 必须包含：我做了什么、结果如何（数据）、踩了什么坑
- 不要求成功——失败复盘比成功案例更珍贵
- 分享后7天内，分享内容整理发布到知识星球

**参会纪律：**
- 不要求全部到场
- 来了就要发言——"我有启发因为XX"也行

### 火种分享会大纲模板

```
🎤 火种分享：[主题]
【结果先行】先告诉大家我做到了什么（数据）
【项目背景】为什么开始做这个
【关键步骤】3-5步，每步有动作有结果
【最大的坑】踩过最惨的一次
【如果再做一次】会怎么改
【你可以试试】给听众的一个可操作的建议
```

## 四、星火值系统 🟡试运行

### 4.1 设计原则

> 星火值是共同体内部的贡献记录，不是货币、股权、债权，也不代表收益承诺。

### 4.2 获取规则

| 行为 | 星火值 | 上限 | 备注 |
|------|--------|------|------|
| 分享实战经验（火种） | +10 | 5次/月 | 知识星球发帖，审核后计入 |
| 共燃项目成功交付（火成） | +20 | — | 团队所有成员共享 |
| 共燃项目失败但有深度复盘 | +10 | — | 失败复盘同样珍贵 |
| 牵线搭桥促成协作（引火） | +5 | 5次/月 | 双方确认 |
| 给火种分享写有质量反馈 | +2 | 10次/月 | 具体回应，非"学到了" |
| 参加活动并发言 | +3 | 3次/月 | 活动现场确认 |
| 带来新星火者（传火） | +20 | — | 对方完成点火 |
| 帮助新人解答问题 | +2 | 10次/月 | 被帮助者确认 |
| 项目无故跳票 | -15 | — | 最严重的违规 |
| 严重违规 | 清零 | — | 一票否决 |

### 4.3 星火值用途

- 活动优先报名
- 项目优先认领
- 内容展示和荣誉
- 核心讨论参与
- 年度荣誉评选

### 4.4 公示

每月1号在知识星球发布星火值TOP30排行榜。

## 五、内容生产 SOP

### 5.1 精华帖标准

每篇精华帖必须包含7要素：

```
1. 星火者名字（真人）
2. 项目名称（一句话说清做了什么）
3. 结果（真实数据）
4. 时间周期
5. 关键步骤（至少3步）
6. 踩过的坑（至少1个）
7. 可复用的经验（至少1条）
```

### 5.2 精华帖审核流程

1. 星火者发帖（标注 #申请精华）
2. 核心星火者3人投票（1票即可进入候选）
3. 主理人终审（把关真实性）
4. 通过 → 进入精华区 + 加星火值10分
5. 未通过 → 私聊反馈修改建议

### 5.3 内容来源（不靠写，靠做）

| 来源 | 处理方式 |
|------|----------|
| 火种分享会 | 分享者准备大纲 → 录音 → AI转文字 → 分享者审核 → 发布 |
| 共燃项目结项 | 点火人填写结项模板 → 核心星火者审核 → 发布 |
| 线下活动 | 每人写3句收获 → 拼成合集 → 发布 |
| 引火成功 | 双方各写100字 → 合并 → 发布 |

### 5.4 星火档案

每季度末，由1位核心星火者轮值整理本季度所有精华帖，输出《星火档案·第X季》。

### 5.5 对外内容分发

| 渠道 | 栏目名 | 内容来源 | 频率 |
|------|--------|----------|------|
| 公众号 | 星火手记 | 精华帖精选+改写 | 每周1篇 |
| 小红书 | 星火笔记 | 精华帖中有画面感的内容 | 每周2-3条 |
| 视频号 | 星火现场 | 线下活动切片 | 有活动就有 |
| 官网 | 星火项目 | 年度最佳项目拆解 | 每年更新 |

## 八、主理人工作手册

### 8.1 时间分配

| 板块 | 占比 | 具体动作 | 频率 |
|------|------|----------|------|
| 方向判断 | 30% | 看趋势、在群内抛话题 | 每天 |
| 连接撮合 | 25% | 引火（私聊撮合）、1v1深度沟通 | 每周5-10次 |
| 内容把关 | 20% | 精华帖审核、星火档案 | 每周2-3小时 |
| 项目孵化 | 15% | 筛选潜力共燃项目，深度参与 | 每月1-2个 |
| 日常运营 | 10% | 群管理、点火仪式、升群评估 | 每天碎片 |

### 8.2 不要做的五件事

1. 不要自己写所有内容——你是策展人，不是主编
2. 不要在群里回答所有问题——把问题抛回给群
3. 不要每次活动都自己上——轮值制是DNA
4. 不要追求完美系统——先跑起来
5. 不要替别人做决定——连接，不是替代

### 8.3 团队配置 🔵计划中

| 阶段 | 人力 |
|------|------|
| 起步期 | 主理人 + 1个助理 |
| 成长期 | 主理人 + 助理 + 1个内容编辑 |
| 成熟期 | 主理人 + 运营助理 + 内容编辑 + 1个BD |

### 8.4 助理工作清单

**每日：** 新人欢迎+引导点火 | 审核星球新帖（看有无违规） | 记录引火/传火事件
**每周：** 发布火种速递 | 更新星火值 | 整理分享会录音
**每月：** 发布星火值TOP30 | 协助清理 | 整理精华帖归档
