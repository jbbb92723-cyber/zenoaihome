# md2wechat 发布提示词

> 用途：把公众号版 Markdown 文稿发布到微信公众号
> 使用时机：公众号版文章写好后，准备发布

---

## 使用说明

`md2wechat` 是一个将 Markdown 转换为公众号格式 HTML 的工具。

发布公众号前，先用这个提示词让 Claude 生成格式规范的 Markdown 文稿，再通过 md2wechat skill 发布。

---

## 公众号文章格式提示词

```
你是赞诺的公众号排版助手。

请把以下内容整理成适合微信公众号发布的 Markdown 格式：

【粘贴公众号版文章内容】

格式要求：
1. 标题用 H1（#）
2. 章节标题用 H2（##）
3. 重点句子用**加粗**标注
4. 段落短，每段不超过4行
5. 适当加空行，公众号阅读需要更多留白
6. 不要用表格（公众号表格显示不好）
7. 不要用代码块
8. 引用或金句用 > 引用格式

末尾固定模块：
---
**关于赞诺**
装修行业从业 16 年，现实派生活建造者。
帮你在装修前，想清楚该想清楚的事。

微信咨询：zanxiansheng2025
---

输出整理好的 Markdown，我来使用 md2wechat 发布。
```

---

## md2wechat 发布流程

1. 打开 Claude 对话
2. 把公众号版.md 的内容，用上面的提示词整理好格式
3. 调用 md2wechat skill，输入：
   - 文章内容（整理后的 Markdown）
   - 标题
   - 封面图（从 媒体资产/封面图/ 下载）
   - 摘要

---

## 常见问题

**图片怎么处理？**
公众号图片需要先上传到公众号素材库，再在 md2wechat 里引用图片 ID。

**标题怎么选？**
公众号标题建议 A/B 测试，准备 2-3 个备选标题。
