# 微信自动回复配置操作步骤

> 前提：zanxiansheng2025 是一个微信公众号（服务号/订阅号），不是个人微信。
> 个人微信不支持自动回复。如果你用的是个人微信，下面的"方案B"适用。

---

## 方案A：公众号自动回复（如果你有公众号）

### 登录后台
微信公众平台 → https://mp.weixin.qq.com → 登录 → 左侧菜单「内容与互动」→「自动回复」

### 1. 被关注回复
粘贴这段：

```
你好，我是赞诺（Zeno）。

做了17年装修，现在专门帮即将签合同的业主审核报价单。

你可以直接发消息说你的情况。

🔍 发「初筛」→ 免费报价风险初筛工具
📋 发「审报价」→ ¥2,500 零加价保障审查
📖 发「文章」→ 装修报价相关文章合集

不急。装修是大事，我们一起看清楚再签。
```

### 2. 关键词回复

点击「关键词回复」→「添加规则」

**规则1：**
- 规则名：报价审核
- 关键词：审报价、报价审核、审核、报价（选"半匹配"）
- 回复内容：粘贴 `OPS-20260711-001_微信自动回复流程.md` 里的「关键词：审报价」段

**规则2：**
- 规则名：免费初筛
- 关键词：初筛、免费、工具
- 回复内容：粘贴对应的「关键词：初筛」段

**规则3：**
- 规则名：文章推荐
- 关键词：文章、内容、学习
- 回复内容：粘贴对应的「关键词：文章」段

### 3. 收到消息回复（默认回复）
当用户发的消息不匹配任何关键词时，自动回复：

```
收到。我会在1-3个工作日内回复。

如果你的问题比较急，可以直接把报价单拍照发过来，
告诉我城市、面积、装修方式（半包/全包）。

或者先试试免费初筛工具：https://zenoaihome.com/tools/quote-check
```

## 方案B：个人微信（没有公众号）

个人微信不支持自动回复。用以下替代方案：

### 快捷回复（微信收藏笔记）
1. 打开微信 → 我 → 收藏 → 新建笔记
2. 粘贴被添加时的欢迎语（见上面）
3. 保存
4. 每次有新好友添加，从收藏里调出笔记，复制粘贴发送

### 朋友圈置顶
发一条朋友圈，置顶：

```
📋 新加我的朋友看这里：

问装修报价相关问题的，直接发报价单照片 + 城市 + 面积 + 装修方式。
或者先用免费初筛工具自己扫一眼：zenoaihome.com/tools/quote-check

不接广告、不拉群、不推销。纯判断。
```

---

## 无论用哪个方案，关键原则

1. **第一句话就让人知道你干什么的**——不要"你好，很高兴认识你"，要"做了17年装修，专门帮人审核报价单"
2. **给明确的下一步**——不发名片，发链接（初筛工具、审核页面）
3. **不催**——"不急"两个字比你想象的重要
4. **回复风格**——直接说话，不用"亲""您好""呢""哦"
