# 禁用表达清单

> 用途：AI 输出内容时，必须检查是否包含以下表达。如果有，必须修改。
> 
> 规则：凡是文章里出现这个清单里的句子/词语/表达方式，一律替换。

---

## 一、AI腔（最常见，最需要注意）

| 禁止使用 | 替换方向 |
|---|---|
| 作为一个 AI，我认为… | 直接给出判断，不说来源 |
| 我无法提供… | 直接说你能做什么 |
| 以下是一些建议… | 直接给建议，不用这种过渡句 |
| 总的来说，我们可以得出… | 直接说结论 |
| 综上所述… | 不用总结腔，直接收尾 |
| 需要注意的是… | 说清楚具体是什么 |
| 值得一提的是… | 直接说重点 |
| 这里有几点重要信息… | 直接给信息 |

---

## 二、营销腔

| 禁止使用 | 替换方向 |
|---|---|
| 闭眼入 | 说清楚适合什么人 |
| 全网最低 | 不做价格保证的说法 |
| 保姆级教程 | 说明文章内容是什么 |
| 建议收藏！| 不用催收藏的话 |
| 转发给你身边需要的人 | 不催转发 |
| 太实用了！| 不夸自己的内容 |
| 干货满满 | 不用这类自我标榜词 |
| 解锁 XXX 的秘密 | 不用「解锁」「秘密」这类词 |
| 这些年我终于明白了… | 励志腔，不用 |

---

## 三、焦虑腔

| 禁止使用 | 替换方向 |
|---|---|
| 你还不知道这些，就晚了！| 直接说内容是什么 |
| 90% 的人都不知道… | 不用这种夸张开头 |
| 现在不看，后悔来不及 | 不催紧迫感 |
| 踩坑之后才后悔... | 可以说「我见过」，但不要用恐吓语气 |

---

## 四、公文腔

| 禁止使用 | 替换方向 |
|---|---|
| 为了满足广大消费者的需求… | 直接说内容 |
| 基于以上分析，可以得出… | 直接说结论 |
| 通过本文，读者将会了解… | 不用这种开场白 |
| 随着社会的发展… | 不用宏观开场 |
| 在当今时代… | 不用时代背景铺垫 |

---

## 五、数字夸张

| 禁止使用 | 替换方向 |
|---|---|
| 100% 有效 | 说具体适合什么场景 |
| 超过 90% 的人… | 如果没有真实数据，不用数字 |
| 省下 XX 万的秘诀 | 不用具体承诺金额，除非有真实案例 |
| 1 天搞定 | 不承诺夸张的时间 |

---

## 六、虚假案例标志词

如果文章里出现以下表达，说明内容可能是 AI 编造的，必须替换为真实案例或删除：

- 某客户反映…
- 有用户表示…
- 据悉…
- 业内人士透露…
- 研究表明…（没有来源）
- 数据显示…（没有来源）
- 专家指出…（泛指专家）

---

## 七、立场模糊词

| 禁止使用 | 替换方向 |
|---|---|
| 这个问题没有标准答案… | 给出你的判断 |
| 每个人情况不同，需要具体问题具体分析… | 说你的观点，同时说清楚适用条件 |
| 仁者见仁，智者见智… | 不用这种逃避判断的话 |

---

## 检查规则

每篇文章发布前，用以下问题自查：

- [ ] 文章里有没有我没有经历过的「案例」？
- [ ] 有没有出现上述禁用词汇？
- [ ] 文章的语气是「我见过」还是「据说」？
- [ ] 有没有夸张的数据或时间承诺？
- [ ] 有没有逃避观点的模糊表达？
