## 核心内容

FIRE = Fact事实 → Interpretation解读 → Reaction反应 → End结果。
SCOUT = Specific具体 / Candid公正 / Objective客观 / Unemotional不带情绪 / Timely及时（不翻旧账）。

装修验收场景：业主说"做工太差"→ Fact：3处不平整超3mm / Interpretation：业主解读为不专业 / Reaction：愤怒不信任 / End：希望返工+道歉。正确的FIRE回应：承认事实→理解解读→接住情绪→给出方案和时间。
