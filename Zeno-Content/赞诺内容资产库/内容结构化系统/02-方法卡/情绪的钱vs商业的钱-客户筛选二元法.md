# 情绪的钱 vs 商业的钱：筛选客户的二元分法

## 一句话

半数以上的人提问不是为了获取答案，而是为不行动寻找一个足够权威的许可。

## 底层机制

```
提问者真实动机 ≠ 字面问题
→ 字面问题 = 行动障碍的表层表达
→ 真实动机 = 为不行动寻找权威背书
→ 如果回答者给出"可以绕过障碍"的答案 → 提问者会换一个理由继续不行动
→ 如果回答者说"确实没办法" → 提问者获得许可，心安理得放弃
```

核心：这类提问的正确答案不是"怎么解决"，而是"你到底想不想做"。

## 二分选择

| 维度 | 赚情绪的钱 | 赚商业的钱 |
|------|-----------|-----------|
| 做法 | 抚慰对方，给台阶下 | 拆穿借口，给行动路径 |
| 客户群 | 不想行动的人 | 想行动但缺方法的人 |
| 客单价 | 必须低 | 可以高 |
| 退款率 | 高（做不成事，必退） | 低（拿到结果，不退） |
| 长期后果 | 耗尽精力，养出一群依赖者 | 筛选出行动者，口碑正向循环 |

## 示例

**提问**：我没有营业执照，怎么注册企业专业号？

- 赚情绪的回答：确实，没有营业执照是注册不了的，你可以先做个人号（提问者获得"不用做了"的许可）
- 赚商业的回答：你可以有。去办一个，或者找代办。

## 对Zeno的业务迁移

### 装修咨询场景的等价问题

| 业主说 | 字面问题 | 真实动机可能是 |
|--------|---------|---------------|
| "我预算不够" | 需要省钱方案 | 不想做决策，想听人说"先别装了" |
| "装修公司说这个价格做不了" | 需要谈判建议 | 想让权威说"那就算了" |
| "我老公/老婆不同意" | 需要说服方法 | 自己也不想做，找个挡箭牌 |
| "再看看吧，不急" | 需要时间 | 永远不会"看好了" |

### 判断标准

业主提问后48小时内是否有行动（发了报价单/去了工地/问了施工方具体问题）。

有行动 → 赚商业的钱。没有 → 要么在免费阶段筛掉，要么只赚情绪的钱（低价内容/工具，不接服务）。
