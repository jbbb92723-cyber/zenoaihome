## 优先级排序

### P1 — 这周先做（高价值 + 大缺口）

| 边界 | 理由 |
|---|---|
| **水电点位** | 金额最大的隐藏加价来源。每多一个点位就是一笔钱。客户最敏感。 |
| **按实际结算** | 七大罪恶之首。七个字可以藏半页纸的坑。 |
| **另计·甲供·暂估** | 直接关联总价是否完整。签约前最后一道防线。 |
| **低价报价** | 最常见的业主误区。"便宜"是最贵的词。 |
| **付款节点** | 钱付完了就没筹码了。每个业主都该知道。 |
| **验收质保** | 装修完了才发现问题——验收标准决定你能追回什么。 |
| **材料型号** | 同品牌不同系列差价一倍。最容易忽略。 |

### P2 — 两周内清掉

| 边界 | 理由 |
|---|---|
| 基础部分总合计 | 有素材，整理即可 |
| 防水 | 素材最多，挑选精华整理 |
| 封窗 | 有一定素材 |
| 垃圾外运 | 有素材，且已有一篇完整文章 |
| 工期顺延 | 影响面较小，但仍有价值 |

## 产出节奏建议

一周做2个P1边界。每个边界一次产出四种格式（用TMP-HubSpoke模板）。
- 两个边界 = 2篇深度文章 + 2条短视频脚本 + 2个小红书图文 + 2页PDF清单

四周做完所有P1（7个边界）= 7篇文章 + 7条视频脚本 + 7个小红书 + 7页PDF → ¥49自查工具包的主体内容就够了。
