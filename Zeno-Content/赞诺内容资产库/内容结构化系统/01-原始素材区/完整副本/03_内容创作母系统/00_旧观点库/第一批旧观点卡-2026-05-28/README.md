## 使用方式

1. 看到一个爆款选题，先在本目录检索相关旧观点。
2. 将相关旧观点分成：可复用 / 已说腻 / 需要回避。
3. 开启素材膨胀时，第一轨基于旧观点扩展，第二轨明确回避旧观点寻找新角度。
