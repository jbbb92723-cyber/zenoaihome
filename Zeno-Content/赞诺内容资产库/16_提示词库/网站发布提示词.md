# 网站发布提示词

> 用途：把网站版文章发布到 zenoaihome.com（Next.js 网站）
> 使用时机：网站版.md 写好后，准备添加到网站代码

---

## 网站文章发布流程

### Step 1：准备文章文件

网站文章放在：`g:\Zenoaihome.com\zeno-site\app\blog\` 目录下

文件命名规则：`[slug].md` 或者在 `data/articles.ts` 里添加数据

### Step 2：文章数据格式

在 `data/articles.ts` 里，每篇文章的数据格式：

```typescript
{
  slug: 'zhuangxiu-yusuan-weishenme-zongchao',
  title: '装修预算为什么总超',
  excerpt: '装修超预算，不是因为你乱花钱，是因为报价单里有一个结构性漏洞...',
  date: '2026-04-20',
  category: '居住与装修',
  readTime: '8 分钟',
  // 其他字段参考现有文章
}
```

### Step 3：发布前检查

使用以下提示词让 Claude 帮你检查：

```
你是赞诺网站的内容质检助手。

请检查以下网站版文章是否符合发布标准：

【粘贴网站版.md 全文】

检查项目：
1. 标题是否包含主关键词？
2. 元描述是否在 150 字符以内？
3. 正文是否有 H2/H3 结构？
4. 是否有 FAQ 区（GEO 问答）？
5. 是否有服务转化入口？
6. 有没有出现禁用词（AI腔/营销腔/夸张数字）？
7. 有没有空洞的泛泛而谈（没有真实案例支撑的观点）？

输出：问题清单 + 修改建议（如果有问题）
或者：「通过，可以发布」
```

---

## 网站发布注意事项

- 图片放在 `public/images/articles/` 目录下
- 图片尺寸建议：封面图 1200×630px
- 内链检查：文章里有没有可以链接到其他文章或服务页的地方
- 发布后更新 sitemap（自动或手动更新 `public/sitemap.xml`）
