# Zeno OS 总首页

> Zeno OS 不是“AI 工作流生产内容”的系统，而是个人认知、内容表达、流量选题、商业项目和 Agent 协作的本地资产系统。

## 核心原则

- 不用 AI 一键生产重要逐字稿。
- AI 负责检索、整理、扩张、反驳、索引和局部辅助。
- 最终判断、审美和关键表达由人完成。
- 可以有可调用方法，但必须足够多，不能只有一个万能模板。

## 项目边界

ZenoAIHome 是 Zeno OS 里的一个项目，不是整个系统本身。

当资料属于装修报价、合同、签约前判断、服务页、网站内容时，进入 `12_项目库/ZenoAIHome` 或沿用现有 ZenoAIHome 项目目录。

当资料属于内容表达、流量、个人成长、商业模式、AI 协作时，进入 Zeno OS 上层目录。
