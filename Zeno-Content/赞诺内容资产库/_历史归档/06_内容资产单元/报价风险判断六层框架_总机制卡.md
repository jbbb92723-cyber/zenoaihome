## 1. 隐藏命题

这 14 篇文章表面讲的是不同的报价风险点——水电按个报价、防水品牌不完整、另计甲供暂估、垃圾外运冲突、口头承诺不落地……

但它们真正反对的是同一个东西：

> 以为报价单上的数字就是最终要付的钱，而看不见数字背后的边界、规则和例外。

更精确地说，它们反对三种默认假设：

1. **总价确定假设** — 以为最后一行的数字是封闭的。实际上它是开放的——另计、暂估、不含、按实结算都在说"这个数后面会变"。
2. **术语无害假设** — 对"按实结算""暂估""甲方认可"等术语不追问。实际上每个词都在转移或悬置一部分责任/费用。
3. **善意默认假设** — "他说了会处理"等于"写进合同了"。实际上口头承诺在施工后变形极快。

## 2. 底层机制

**报价风险的本质不是"有人在骗你"。是信息在报价单上的分布不对等。**

一份装修报价包含六层信息。大多数业主只看前两层：

| 层次 | 内容 | 业主通常 |
|---|---|---|
| 1. 项目名称 | "卫生间防水""水电改造" | ✅ 扫一眼 |
| 2. 单价/总价 | "15个×160元""基础部分总合计" | 🔍 仔细看 |
| 3. 计价单位 | "按个""按米""按平方""按项" | ⚠️ 很少追问 |
| 4. 范围边界 | 含不含拆旧/含不含找平/防水做到多高 | 🔴 最常漏 |
| 5. 备注/例外 | "另计""暂估""不含""以现场为准""按实结算" | 🔴 最容易跳过 |
| 6. 责任规则 | 增项谁确认/减项谁承担/争议怎么核算 | 🔴 几乎不看 |

风险就藏在第 3-6 层和 1-2 层之间的**不一致**里：

- 项目名称写了"防水"，范围边界只覆盖卫生间不覆盖厨房 → **品牌不等于完整**。
- 单价写了 160 元/个，但计价单位"个"的定义不清 → **按个报价不等于闭口价**。
- 总价写了 12 万，备注写了"按实际增减结算" → **总价不等于最终价**。
- 明细写了"垃圾外运 1 车"，备注写了"不含垃圾外运" → **前后冲突**。

核心因果链：

```text
业主只看第 1-2 层（名称+价格）
→ 第 3-6 层（单位+边界+备注+规则）被跳过
→ 签合同
→ 施工中第 3-6 层的信息逐一兑现
→ 每一项都"早就写在那里了"，但业主没看到
→ 感觉被坑，但合同上确实有
→ 纠纷
```

反过来：

```text
业主逐层检查（1-6 层）
→ 发现"个"没定义、"暂估"没上限、"不含"没说后果
→ 签约前把定义、上限、规则全补齐
→ 签合同
→ 施工中变化仍然可能发生，但每次变化都有规则可依
→ 不会"突然发现多了一笔钱"
```

## 3. 六层框架

### 第一层：项目有没有漏
报价单上没写的项目，不代表不需要做。漏项不是"省了"，是"还没算进来"。

→ 拆除、垃圾清运、成品保护、材料上楼、税费、管理费——这六项最容易漏。

### 第二层：数量有没有少算
写了不等于算够了。一车垃圾够不够？一个插座够不够？一个回路够不够？

→ 问预估依据，问超出规则。

### 第三层：材料型号有没有写清
品牌不等于规格。写了"某品牌防水"不代表防水完整。写了"断桥铝"不代表厚度、玻璃、五金都定好了。

→ 品牌、型号、规格、替换规则——四样缺一不可。

### 第四层：计价单位有没有定义
"按个""按米""按平方""按项"——每个单位都可能被不同理解。

→ 一个"个"到底是点位、回路、水口还是包含开槽穿线？

### 第五层：备注和明细有没有冲突
明细写了"含"，备注写了"不含"——这是报价前后不一致。

→ 签约前必须改到一致，不能带着冲突签。

### 第六层：增项和变化的规则有没有前置
变化本身不是问题。没有规则的变化才是问题。

→ 增项谁确认、怎么确认、减项怎么扣、争议怎么算——签约前写好。

## 4. 跨越 14 篇的术语词典

| 术语 | 含义 | 风险 | 签约前追问 | 对应文章 |
|---|---|---|---|---|
| **另计** | 当前总价不含此项，后面单独算 | 业主以为"含了"，实际没含 | 为什么另计？按什么单位/单价算？什么情况下触发？ | 另计甲供暂估 |
| **甲供** | 业主自己买，施工方可能只管安装或不负责 | 测量/下单/运输/售后责任不清 | 谁测量？谁下单？不合适谁负责？安装辅料谁出？ | 另计甲供暂估 |
| **暂估** | 当前金额只是预估，最终调整 | 预估和结算差距大，没有上限 | 怎么估的？按什么标准结算？超出多少要确认？低于暂估退不退？ | 另计甲供暂估 |
| **不含** | 明确不包含的内容 | 业主没看到"不含"，以为全包了 | 不含的这项影响什么？需要做吗？多少钱？ | 垃圾外运冲突 |
| **按实结算** | 最终按实际发生量结算 | 开了口子但没有规则 | 单价表有没有？上限有没有？增项谁确认？减项怎么扣？ | 按实际增减结算 |
| **口头承诺** | 沟通时说过的话 | 施工后变形，"说过"≠"写了" | 把 5 类承诺写进合同：价格/工期/材料/增项/售后 | 5个口头承诺 |
| **基础部分总合计** | 当前已列项目的合计 | 不是最终闭口价 | 哪些未包含？哪些按实结算？哪些另计？ | 基础部分总合计 |
| **品牌（仅写品牌）** | 写了材料牌子没写型号规格 | 同一品牌不同型号差价大，替换无约束 | 型号是什么？替换规则是什么？替代品差价谁补？ | 防水品牌/材料品牌 |
| **按个/按米（计价单位模糊）** | 计价单位有歧义 | "个"可能是点位/回路/水口，定义不同价格差几倍 | 这个"个/米"具体指什么？含了哪些工序？ | 水电按个报价 |

## 5. 一句话

> 报价单上最危险的不是数字大。是你不确定后面还有多少数字没进来。
