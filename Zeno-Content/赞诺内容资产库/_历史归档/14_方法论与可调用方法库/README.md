# 方法论与可调用方法库

> 这里不用“套路”这个词。更准确的叫法是：**可调用方法**、**方法组件**、**判断框架**、**操作法**。

它们不是自动写稿模板，而是在合适场景调用的思考工具。

## 分类

- [[内容表达方法]]：写作、表达、选题展开、标题开头辅助。
- [[流量选题方法]]：爆款选题识别、旧观点检索、新角度生成、平台复盘。
- [[商业判断方法]]：产品化、需求验证、转化路径、定价判断。
- [[项目推进方法]]：计划、复盘、执行、风险检查。

## 使用原则

1. 方法可以复用，但不能只剩一个模板。
2. 方法负责把思考推到更好的起点，不负责替人完成最终表达。
3. 重要逐字稿由人写，AI 只做局部辅助和扩张。
4. 每个方法都要说明：适用场景、输入、步骤、产物、不适合做什么。
