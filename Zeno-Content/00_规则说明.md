# Zeno-Content 规则说明

## 定位

这里存放 ZenoAIHome 的内容资产、文案、素材、项目文档、笔记和知识库资料。

## 可以放这里

- 网站文案草稿、文章、选题、标题、发布稿。
- Obsidian 笔记、知识卡片、内容结构化资料。
- 品牌资料、运营资料、用户研究、服务说明文档。
- 图片、视频、音频、PDF、表格等非程序素材。
- 项目方案、审计记录、阶段性总结、人工检查清单。

## 不放这里

- Next.js、React、TypeScript、API、后台管理等程序源码。
- `node_modules`、`.next`、构建输出、依赖缓存。
- 本地环境变量、密钥、数据库连接配置。
- 自动化脚本、工具配置、浏览器缓存、运行日志。

## 归类原则

如果一个文件主要回答“说什么、给谁看、作为素材如何复用”，优先归入这里。

如果文件既像文档又影响程序运行，例如 API 文档、部署说明、开发约定，先放入 `待归类文件夹`，等待人工确认。
