# 中英文切换策略

> 适用于 Zeno 赞诺 (zenoaihome.com)

## 基本原则

1. **中文是默认主站**。所有内容、资料、服务、转化入口以中文站为核心。
2. **英文不是自动翻译**。英文内容由人工撰写或精心改写，不做全站机器翻译。
3. **英文只放精选内容**。第一阶段包括：首页、关于、专题概览、精选文章占位。
4. **中文站负责主要信任、资料、服务和转化**。
5. **英文站负责国际化表达**，承接 YouTube 等国际渠道流量，沉淀长期思想内容。

## 技术实现

- 中文页面路径：`/`、`/about`、`/blog`、`/topics` 等
- 英文页面路径：`/en`、`/en/about`、`/en/articles`、`/en/topics` 等
- 语言切换组件内置于 Header，显示为「中文 / EN」
- 切换逻辑：尝试跳转到对应语言的同路径页面，若不存在则回退到首页

## 后续扩展

1. 逐步新增英文文章（手动编写，非机翻）
2. 英文文章存放在 `app/en/articles/[slug]/page.tsx`
3. 可考虑增加 `app/en/resources/page.tsx`（精选资料英文版）
4. 不急于做完整双语——质量优先于覆盖率

## 不做的事

- 不做全站自动翻译
- 不使用 i18n 框架做路由重写（保持简单 App Router 结构）
- 不在同一个页面内切换语言（每种语言有独立页面）
