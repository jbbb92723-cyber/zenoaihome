## 领取闭环设计

```
用户看到文章/专题
  └→ 文章末尾引导到资料库 /resources
       └→ 资料卡片说明"公众号回复XXX领取"
            └→ 关注公众号 → 自动回复资料
                 └→ 如需深度帮助 → 查看服务 /services → 微信咨询
```
