# G2 · 表达 DNA Agent 输出

## 碎片表达与语言风格

### 高频词汇与句式（语言指纹）
**高频短句**：
- "说白了" / "说穿了" / "是两码事" / "一回事"
- "涨粉就是不务正业"
- "智商决定收入上限，商业模式决定收入下限"
- "执行力就是业务门槛"
- "把'赛道''行业'这俩词从脑子里删掉"

**句式特征**：
- 极度口语化，接近说话而非写作
- 频繁使用引号包裹概念（"干货""副业""赛道"），暗示对这些概念的批判性审视
- 短句为主（15-30字），极少长句
- 先下断言，再解释或举例
- 喜欢用"XX + YY"结构制造概念张力（"强唯物主义+实践验证主义"）
- 反问句频繁使用，制造对话感

### 幽默风格
- 冷幽默 + 自嘲：主动列出自己的不自洽之处作为"批判 don 哥的角度"
- 反讽：对商业世界的荒谬现象采取"翻译成大白话"的讽刺方式
- 几乎不使用网络热梗或流行语

### 自我叙事模式
- 不讲述自己的成功故事（因为匿名，无法讲个人经历）
- 通过"我发现了什么"而非"我做到了什么"来建立权威
- 引用"8,000+付费问题""300万字直播文稿""13,000+条推文"等数据而非个人履历来证明可信度
- 把自己定位为"商业哲学爱好者"而非"创业者/成功人士"

### 情绪表达习惯
- **愤怒时**：措辞更简洁更尖锐，使用反问句（"这跟印钞机有什么区别？"）
- **兴奋时**：频繁使用"说穿了""说白了"，语速加快，类比增多
- **脆弱时**：极罕见，唯一可见的是列出"不自洽之处"时的自我暴露——这本身可能是一种防御机制
- **平静时**：用德鲁克/管理学语言，回归系统性分析

### 争议立场与边界感
- 反"干货"：认为这个词是伪概念，制造伪交流
- 反"副业"：认为是致贫行为
- 反"涨粉焦虑"：认为关注变现效率而非粉丝规模
- 反"流量技巧"：认为技巧驱动的流量无法转化为付费
- 反"模糊概念"：认为"赛道""行业"等词阻碍清晰思考

### 确定性风格
- 对自己的核心框架（商业诊断方法论）高度确定
- 对超出框架的问题不回避，但会"消解问题"而非强行回答
- 对自己能力边界的表达："这是我能回答的/这是我回答不了的"

### 来源标注
- 来源类型：网络搜索（本人发布的推文/播客/文章）
- 可信度：高
