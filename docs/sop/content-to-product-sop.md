# 内容到产品 SOP

> 每篇内容都必须连接到：栏目、用户痛点、产品入口、网站页面、短视频选题、资料/工具/服务。

---

## 1. 选题阶段

- [ ] 来源：从灵感收集箱 / 客户问题 / 搜索词 / 评论区中提取
- [ ] 归类：确定属于五大板块中的哪一个
- [ ] 任务标注：信任型 / 流量型 / 转化型 / 产品型 / 精神型
- [ ] 产品连接：确定文章结尾导向哪个产品/服务/页面
- [ ] 搜索验证：用 SEO 关键词地图确认搜索需求存在
- [ ] 短视频切口：确认能否拆出 1-3 个短视频选题

## 2. 写作阶段

- [ ] 结构：问题 → 经历 → 洞察 → 行动
- [ ] 口吻：按 00_定位系统/表达风格.md 要求
- [ ] 长度：800-2000 字（流量型偏短，信任型可长）
- [ ] excerpt：2 句话——第一句切痛点，第二句给方向
- [ ] CTA 段：结尾导向段（不硬推，有逻辑过渡）

## 3. 标题优化阶段

- [ ] 主标题：口语化 + 有判断 + 有悬念或反常识
- [ ] SEO 标题：metadata title 含核心搜索词
- [ ] 禁用检查：不用"震惊""必看""99%的人不知道"（参考 00_定位系统/禁用表达.md）

## 4. 网站发布阶段

- [ ] 数据文件：添加到 data/articles.ts（id/slug/category/tags/date 齐全）
- [ ] 内链：在相关文章中添加关联（至少 2 篇）
- [ ] CTA 组件：确认 blog/[slug] 底部 CTA 覆盖该 category
- [ ] 构建：`npm run build` 通过
- [ ] 部署：push 到 main 触发 Vercel 部署

## 5. 公众号改写阶段

- [ ] 改写：用 06_提示词库/平台改写提示词.md 调整口吻
- [ ] 排版：用 md2wechat 工具排版
- [ ] CTA：文末加引导（关注/回复关键词/查看原文）
- [ ] 记录：在 07_发布复盘/发布记录.md 记录日期和数据

## 6. 短视频拆解阶段

- [ ] 切口：从文章中提取 1 个反常识/具体场景作为 hook
- [ ] 脚本：30-60 秒——hook → 核心观点 → 行动引导
- [ ] 引导：结尾导向主页链接/关注/评论区
- [ ] 发布：抖音/小红书/视频号

## 7. 产品导流阶段

- [ ] 确认：关联产品/服务是否已上线、链接可用
- [ ] 邮件：注册用户的邮件序列是否包含相关产品推荐
- [ ] 埋点：文章 → 产品页的点击转化是否可追踪

## 8. 复盘阶段

- [ ] 数据：7 天后看 UV、停留时间、CTA 点击率
- [ ] 迭代：表现好的加内链、推首页、做专题
- [ ] 产品化：反复被搜索的主题考虑做对应产品
- [ ] 记录：更新 07_发布复盘/发布记录.md
