> 标题：我看报价单只看三行

---

今天又看了一份报价。南宁青秀区一个 112 平的房子，三室两厅。业主发给我整本 PDF，37 页。

我只看了三行。

第一行：水电。写了「按实结算，暂估 2.3 万」。没有写单价。没有写计量规则。没有写上限。关掉 PDF，我给业主发了五个字：「这一行不能签。」

不是 2.3 万对不对的问题。是「暂估」两个字把不确定性全都甩在了业主这一边。施工方可以做完之后说「你家水路确实复杂，超支了 7 万」，你没有反驳的依据。因为暂估=我不知道要花多少。你不知道要花多少，为什么要签？

2018 年我自己就被这个坑过。暂估 1.2 万，结算 3.7 万，多了整整两倍。其中最离谱的一笔：一根排水管，原来就在原户型里的，不需要新走，施工方把它报在「新增排水」里收了 4000 块。我蹲在工地地上用手电筒照了半天才确认。

从那以后，每一个拿到我面前的报价单，水电这一行我在找三个字：**上限呢**。

第二行：付款。合同写首付 60%。60%。我见过首付 50% 的合同，业主三个月催不动一次整改——钱已经出去了大半，施工方不急，业主每一次催都像是在求人家。

业主说：「可是他们说这是标准付款方式。」不是。「标准」两个字是放在合同前面让人放松警惕的。你问的是「这个比例你接不接受」，他回的是「大家都是这样的」。这不是回答。这是话术。

我问业主：尾款多少？她说 5%。我说，5% 的尾款，你拿什么让人回来改东西？尾款就是你手里最后的牌，这张牌必须够大，大到对方会真的回来。

第三行：验收。水电验收那个格子写的：「按相关标准验收合格」。没有写什么标准。没有写谁签字。没有写验收不过怎么处理。

四年前一个客户，水电验收时去工地看了一眼——水管有，电线有，觉得行了。入住半年后水压不稳。回头找施工方：「当时你验收签了字的。」签了。但签字的人不知道要测水压。

我现在告诉每个业主：验收不是看一圈，是按标准测一遍。水压加压 30 分钟不回落叫合格。闭水 48 小时不漏叫合格。每一道工序做完，要有人签字写日期。签之前去网上打印一份验收清单。你不是装修专业，你是住这个房子的人——专业的事交给标准，你的事是确认标准被执行了。

---

三行看完。十分钟。

这份报价最好的结果不是「谈一谈」，是业主换一家施工方、在签合同之前把上限、付款比例、验收标准堵上。如果再遇到一个说「大家都是这样的」的施工方，她知道怎么回了：「不用管大家怎么样的，我的合同里写我的。」
