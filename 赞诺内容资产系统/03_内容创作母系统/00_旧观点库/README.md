## 使用原则

1. 爆款选题来了，先检索旧观点，而不是直接写。
2. 旧观点可以复用，也可以作为禁用清单，逼出新角度。
3. 旧观点卡不是最终稿，只是素材膨胀和人工写稿的起点。
4. 如果某个观点已经说腻，就不要重复包装，改走“回避旧观点”的第二轨。
